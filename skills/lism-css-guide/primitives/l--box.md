# l--box / `<Box>`

コンテンツをグループ化するだけのシンプルなクラス。汎用的な箱として、パディング・ボーダー・背景色などの指定に使います。

## 基本情報

- クラス名: `l--box`
- コンポーネント: `<Box>`
- 公式ドキュメント: https://lism-css.com/docs/primitives/l--box.md

**使い方・コード例については、公式ドキュメントを参照すること。**
