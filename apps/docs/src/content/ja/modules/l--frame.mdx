import { Box, Flow, Flex, Media, Layer, Frame, Center, Columns, Inline } from 'lism-css/astro';
import { DummyText } from '@lism-css/ui/astro';


`l--frame` は、直下のメディア要素（`img`,`video`,`iframe`）を 自身のサイズに合わせて表示するレイアウトモジュールです。


<PreviewTitle>CSS</PreviewTitle>
<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/modules/layout/_frame.scss' />


## 対応コンポーネント

`lism-css` では`<Frame>`コンポーネントを配布しています。

### Import

<ImportPackage component='Frame' />

### Props

| プロパティ | 説明 |
|---|---|
| `ar` | フレームの比率を指定できます。|
{/* | `objectPosition` | 直下のメディア要素のobject-positionを指定できます。 | */}

<HelpText>(`ar` 自体はどのコンポーネントにも指定できる CSS Props です。)</HelpText>

### ar はどのコンポーネントにも指定できます

:::note
::title[arで指定できる値について]
`ar`(`aspect-ratio`のCSS Props)は他のコンポーネントでも指定できる汎用的なプロパティです。  
[使用できる値についてはこちらをご覧ください。](/docs/prop-class/#aspect-ratio)
:::
