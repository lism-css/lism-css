### センター寄せレイアウト

`<Stack ai="center">`( `.l--stack.-ai:center` )の組み合わせで、Every Layout で言うところの **"内在的な中央寄せ"** （コンテンツが短い時だけ中央寄せ）にできます。

<Preview>
  <PreviewTitle>内在的な中央寄せ</PreviewTitle>
  <PreviewArea resize>
    <Stack ai="center" g="20" p="30" className="has--itemGuide">
      <h3 role="presentation">見出し</h3>
      <DummyText length="s" />
      <DummyText length="l" />
    </Stack>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
  ```jsx 'ai="center"'
  <Stack ai="center" g="20" p="30">
    <h3>見出し</h3>
    <p>...</p>
    <p>...</p>
  </Stack>
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
  ```jsx "-ai:center"
  <div class="l--stack -ai:center -p:30 -g:20">
    <h3>見出し</h3>
    <p>...</p>
    <p>...</p>
  </div>
  ```
  </PreviewCode>
</Preview>
