import { TileGrid, Box, Center } from 'lism-css/astro';


`l--tileGrid` は、`cols`（列数）と`rows`（行数）を指定して、**均等なタイル型グリッドレイアウト**を構成するためのレイアウトモジュールです。

内部的には `grid-template: repeat(rows, minmax(0, 1fr)) / repeat(cols, minmax(0, 1fr))` を使用しています。

<ImportPackage component="TileGrid" />

### 専用Props

| プロパティ | 説明 | デフォルト値 |
|---|---|---|
| `cols` <PropBadge type="cssvar">`--cols`</PropBadge> | 列数を指定します。ブレイクポイント指定も可能です。 | `1` |
| `rows` <PropBadge type="cssvar">`--rows`</PropBadge> | 行数を指定します。ブレイクポイント指定も可能です。 | `1` |

### ブレイクポイント指定

`cols`、`rows` はそれぞれブレイクポイント指定が可能です。

<Preview>
  <PreviewTitle>例</PreviewTitle>
  <PreviewArea resize p="15">
    <TileGrid cols={['2', '3']} rows="4" min-h="50svh" gaf="row dense">
      <Box gc={['1 / -1', 'span 2']} gr={['span 2', '1 / -1']} p="20" bgc="base-2">A</Box>
      <Box p="20" bgc="green:20%">B</Box>
      <Box p="20" bgc="blue:20%">C</Box>
      <Box p="20" bgc="red:20%">D</Box>
      <Box p="20" bgc="purple:20%">E</Box>
    </TileGrid>
