import { DummyText } from '@lism-css/ui/astro';

`<Text>`は、`<p>`タグを出力するLismコンポーネントです。

:::info
`<Text>` は `<Lism as="p">` のエイリアスです。全ての [Lism Props](/docs/core-components/lism-props/) が使用できます。
`as` prop で出力タグを変更できます: `p`(デフォルト) / `div` / `blockquote` / `address` / `figcaption` / `pre`
:::

## Import

<ImportPackage component="Text" />
