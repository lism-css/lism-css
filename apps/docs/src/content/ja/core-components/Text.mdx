import { Text, Lism, Dummy } from "lism-css/astro";

`<Text>`は、`<p>`タグを出力するLismコンポーネントです。

:::info
`<Text>` は `<Lism as='p'>` のエイリアスです。全ての [Lism Props](/docs/core-components/Lism/) が使用できます。  
`as` prop で出力タグを変更できます: `p`(デフォルト) / `div` / `blockquote` / `address` / `figcaption` / `pre`
:::

## Import

<ImportPackage component='Text' />


## 基本的な使い方

<Preview>
  <PreviewTitle>`<Text>`の出力例</PreviewTitle>
  <PreviewArea p='20'>
    <Text><Dummy as="span" /></Text>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>

    <Text>Lorem ipsum text...</Text>

  <PreviewCode slot='tab' label='HTML'>
