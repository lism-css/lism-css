`<Link>`は、`<a>`タグを出力するLismコンポーネントです。

:::info
`<Link>` は `<Lism as="a">` のエイリアスです。全ての [Lism Props](/docs/core-components/Lism/) が使用できます。
:::

## Import

<ImportPackage component='Link' />
