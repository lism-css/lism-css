`<Group>`は、`<div>`タグを出力するLismコンポーネントです。  
要素をグルーピングするための汎用コンテナとして使用します。

:::info
`<Group>` は `<Lism as="div">` のエイリアスです。全ての [Lism Props](/docs/core-components/Lism/) が使用できます。  
`as` prop で出力タグを変更できます: `div`(デフォルト) / `section` / `article` / `figure` / `nav` / `aside` / `header` / `footer` / `main` / `fieldset` / `hgroup`
:::

:::note[`<Lism>` との違い]
`<Lism>` もデフォルトで `<div>` を出力しますが、`<Group>` を使うことで「これは要素をグルーピングするコンテナである」という意図がコードから明確になります。
:::
