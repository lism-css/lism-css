import { List, ListItem } from "lism-css/astro";

`<List>`は`<ul>`タグを、`<ListItem>`は`<li>`タグを出力するLismコンポーネントです。

:::info
`<List>` は `<Lism as="ul">` のエイリアス、`<ListItem>` は `<Lism as="li">` のエイリアスです。全ての [Lism Props](/docs/core-components/lism-props/) が使用できます。
`<List>` の `as` prop: `ul`(デフォルト) / `ol` / `dl`  
`<ListItem>` の `as` prop: `li`(デフォルト) / `dt` / `dd`
:::

## 番号付きリスト

`as="ol"` を指定すると、`<ol>`タグとして出力されます。

<Preview>
  <PreviewTitle>`<List as="ol">`の出力例</PreviewTitle>
  <PreviewArea p="20">
    <List as="ol">
      <ListItem>リストアイテム 1</ListItem>
      <ListItem>リストアイテム 2</ListItem>
      <ListItem>リストアイテム 3</ListItem>
    </List>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <List as="ol">
      <ListItem>リストアイテム 1</ListItem>
      <ListItem>リストアイテム 2</ListItem>
      <ListItem>リストアイテム 3</ListItem>
    </List>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html
    <ol>
      <li>リストアイテム 1</li>
      <li>リストアイテム 2</li>

    </ol>
    ```
  </PreviewCode>
</Preview>


## 定義リスト

`<List as="dl">` と `<ListItem as="dt">`, `<ListItem as="dd">` を組み合わせて、定義リストを作成できます。

<Preview>
  <PreviewTitle>`<List as="dl">`の出力例</PreviewTitle>
  <PreviewArea p="20">
    <List as="dl">
      <ListItem as="dt" fw="bold">
        <Fragment>用語 1</Fragment>
      </ListItem>
      <ListItem as="dd">用語 1 の説明文です。</ListItem>
      <ListItem as="dt" fw="bold">
        <Fragment>用語 2</Fragment>
      </ListItem>
      <ListItem as="dd">用語 2 の説明文です。</ListItem>
    </List>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <List as="dl">
      <ListItem as="dt" fw="bold">
        用語 1
      </ListItem>
      <ListItem as="dd">用語 1 の説明文です。</ListItem>
      <ListItem as="dt" fw="bold">
        用語 2
      </ListItem>
      <ListItem as="dd">用語 2 の説明文です。</ListItem>
    </List>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html
    <dl>
      <dt class="-fw:bold">用語 1</dt>
      <dd>用語 1 の説明文です。</dd>
      <dt class="-fw:bold">用語 2</dt>
      <dd>用語 2 の説明文です。</dd>
    </dl>
    ```
  </PreviewCode>
</Preview>
