セマンティックラッパーは、`<Lism as='tag'>` のエイリアスとして機能する名前付きコンポーネント群です。  
コード上でHTML要素の意図を明確にするために使用します。

:::info
すべてのセマンティックラッパーで [Lism Props](/docs/core-components/Lism/) が使用できます。
:::


## Import

<ImportPackage component='Text, Inline, Group, Heading, Link, List, ListItem' />


## 一覧

| コンポーネント | デフォルト `as` | 用途 | `as` で指定可能なタグ |
|---|---|---|---|
| `Text` | `p` | テキストコンテンツ系ブロック要素 | `div` / `blockquote` / `address` / `figcaption` / `pre` |
| `Inline` | `span` | 文中要素（phrasing content） | `em` / `strong` / `small` / `code` / `time` / `i` / `b` / `mark` / `abbr` / `cite` / `kbd` |
| `Group` | `div` | セマンティックな構造ラッパー | `section` / `article` / `figure` / `nav` / `aside` / `header` / `footer` / `main` / `fieldset` / `hgroup` |
| `Heading` | `h2` | 見出し（`level` prop で h1〜h6） | — |
| `Link` | `a` | リンク | — |
| `List` | `ul` | リスト | `ol` / `dl` |
| `ListItem` | `li` | リストアイテム | `dt` / `dd` |

## Inline

テキストの一部にスタイルを適用したい場合などに使用します。

<Preview>
  <PreviewTitle>`<Inline>`の出力例</PreviewTitle>
  <PreviewArea p='20'>

  <PreviewCode slot='tab' label='JSX'>

    ```jsx
    <Text>これは<Inline fw="bold" c="blue">強調された</Inline>テキストです。</Text>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <p>これは<span class="-fw:bold -c" style="--c:var(--blue)">強調された</span>テキストです。</p>
    ```
  </PreviewCode>
</Preview>

## Group

`<Group>`は、`<div>`タグを出力するLismコンポーネントです。  
要素をグルーピングするための汎用コンテナとして使用します。

:::note[`<Lism>` との違い]
`<Lism>` もデフォルトで `<div>` を出力しますが、`<Group>` を使うことで「これは要素をグルーピングするコンテナである」という意図がコードから明確になります。
:::

<Preview>
  <PreviewTitle>`<Group>`の出力例</PreviewTitle>
  <PreviewArea p='20'>
    <Group p="20" bgc="base-2" bdrs="10">
      <Dummy />
    </Group>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Group p="20" bgc="base-2" bdrs="10">
      Lorem ipsum text...
    </Group>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <div class="-p:20 -bgc:base-2 -bdrs:10">
      Lorem ipsum text...
    </div>
    ```
  </PreviewCode>
</Preview>

## Heading

`<Heading>`は、見出しタグ（`<h1>`〜`<h6>`）を出力するLismコンポーネントです。  
`level` プロパティで見出しレベルを指定できます。

| プロパティ | 型 | デフォルト | 説明 |
|---|---|---|---|
| `level` | `'1' \| '2' \| '3' \| '4' \| '5' \| '6'` | `'2'` | 見出しレベルを指定します。 |

<Preview>
  <PreviewTitle>`<Heading>`の出力例</PreviewTitle>
  <PreviewArea p='20'>
    <Heading level="2">見出しテキスト</Heading>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Heading level="2">見出しテキスト</Heading>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <h2>見出しテキスト</h2>
    ```
  </PreviewCode>
</Preview>


### 見出しレベルの指定

`level` プロパティで `<h1>` 〜 `<h6>` を出し分けることができます。

<Preview>
  <PreviewTitle>`level` によるレベルの指定例</PreviewTitle>
  <PreviewArea p='20'>
    <Stack g="10">
      <Heading level="1" fz="2xl">Heading Level 1</Heading>
      <Heading level="2" fz="xl">Heading Level 2</Heading>
      <Heading level="3" fz="l">Heading Level 3</Heading>
      <Heading level="4" fz="m">Heading Level 4</Heading>
    </Stack>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Heading level="1" fz="2xl">Heading Level 1</Heading>
    <Heading level="2" fz="xl">Heading Level 2</Heading>
    <Heading level="3" fz="l">Heading Level 3</Heading>
    <Heading level="4" fz="m">Heading Level 4</Heading>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <h1 class="-fz:2xl">Heading Level 1</h1>
    <h2 class="-fz:xl">Heading Level 2</h2>
    <h3 class="-fz:l">Heading Level 3</h3>
    <h4 class="-fz:m">Heading Level 4</h4>
    ```
  </PreviewCode>
</Preview>

### 定義リスト

`<List as="dl">` と `<ListItem as="dt">`, `<ListItem as="dd">` を組み合わせて、定義リストを作成できます。

<Preview>
  <PreviewTitle>`<List as="dl">`の出力例</PreviewTitle>
  <PreviewArea p='20'>
    <List as="dl">
      <ListItem as="dt" fw="bold">用語 1</ListItem>
      <ListItem as="dd">用語 1 の説明文です。</ListItem>
      <ListItem as="dt" fw="bold">用語 2</ListItem>
      <ListItem as="dd">用語 2 の説明文です。</ListItem>
    </List>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <List as="dl">
      <ListItem as="dt" fw="bold">用語 1</ListItem>
      <ListItem as="dd">用語 1 の説明文です。</ListItem>
      <ListItem as="dt" fw="bold">用語 2</ListItem>
      <ListItem as="dd">用語 2 の説明文です。</ListItem>
    </List>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <dl>
      <dt class="-fw:bold">用語 1</dt>
      <dd>用語 1 の説明文です。</dd>
      <dt class="-fw:bold">用語 2</dt>
      <dd>用語 2 の説明文です。</dd>
    </dl>
    ```
  </PreviewCode>
</Preview>
