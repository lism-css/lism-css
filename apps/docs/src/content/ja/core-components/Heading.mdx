import { Heading, Stack } from "lism-css/astro";

`<Heading>`は、見出しタグ（`<h1>`〜`<h6>`）を出力するLismコンポーネントです。  
`level` プロパティで見出しレベルを指定できます。

:::info
`<Heading>` は `<Lism as="h{level}">` のエイリアスです。全ての [Lism Props](/docs/core-components/lism-props/) が使用できます。
:::

## Props

| プロパティ | 型 | デフォルト | 説明 |
|---|---|---|---|
| `level` | `"1"`~ `"6"` | `"2"` | 見出しレベルを指定します。 |


## Import

<ImportPackage component="Heading" />

<Preview>

## 見出しレベルの指定

`level` プロパティで `<h1>` 〜 `<h6>` を出し分けることができます。

      <Heading level="2">Heading Level 2</Heading>
      <Heading level="3">Heading Level 3</Heading>
      <Heading level="4">Heading Level 4</Heading>
    </Stack>

  <PreviewCode slot="tab" label="JSX">


</Preview>
