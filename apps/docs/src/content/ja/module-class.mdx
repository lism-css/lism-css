Lismでは、次のようなモジュールクラスを使います。

- `is--{module}`: 複数の種類のモジュールに対してつけ外しできるような汎用性の高い状態モジュール群。
- `l--{module}`: レイアウトの構成単位となるようなモジュール群。
- `a--{module}`: レイアウトの最小単位となるモジュール群。
- `c--{module}`: その他のより具体的な役割を持つコンポーネント群。（他のクラスとの組み合わせを前提にした設計になっているものが多い。）


基本的に、State Modulesを除いた各レイヤーの他モジュールクラスとの複数併用はできません。  
たとえば、`.l--`と`.c--`は併用できますが、`.l--XXX`と`.l--YYY`、`.c--XXX`と`.c--YYY`は同一属性内では併用しないものとします。


## State Modules

要素に対して汎用的につけ外しできる状態モジュール群です。

| クラス | 概要 |
|---|---|
| <ModLink slug="is--container" /> | コンテナクエリの基準要素を定義する |
| <ModLink slug="is--wrapper" /> | コンテンツ幅を制御するラッパー要素 |
| <ModLink slug="is--layer" /> | 親要素に重ねるオーバーレイレイヤー |
| <ModLink slug="is--boxLink" /> | ボックス全体をリンク化する |
| <ModLink slug="is--vertical" /> | 縦書きモードにする |
| `is--coverLink` | 親要素全体に広がるクリック領域を持つリンク（→ <ModLink slug="is--boxLink" /> 参照） |
| `is--side` | `l--sideMain` の side 要素用（→ <ModLink slug="l--sideMain" /> 参照） |
| `is--skipFlow` | `l--flow` 内で次の兄弟との余白を打ち消す（→ <ModLink slug="l--flow" /> 参照） |


## Layout Modules

レイアウトの構成単位となるモジュール群です。

| クラス | 概要 |
|---|---|
| <ModLink slug="l--box" /> | コンテンツをグループ化するシンプルなモジュール |
| <ModLink slug="l--flow" /> | 子要素間の余白を自動管理する flow-root モジュール |
| <ModLink slug="l--frame" /> | 直下のメディア要素を自身のサイズに合わせて表示 |
| <ModLink slug="l--flex" /> | Flexbox でコンテンツを配置 |
| <ModLink slug="l--cluster" /> | 横方向に並べて自動折り返し |
| <ModLink slug="l--stack" /> | 要素を縦方向に配置 |
| <ModLink slug="l--sideMain" /> | サイドとメインの 2 カラムレイアウト |
| <ModLink slug="l--grid" /> | CSS Grid でコンテンツを配置 |
| <ModLink slug="l--tileGrid" /> | 均等な行列グリッド（タイル型レイアウト） |
| <ModLink slug="l--center" /> | 上下左右中央揃えで配置 |
| <ModLink slug="l--columns" /> | ブレイクポイントごとに列数を指定するカラムレイアウト |
| <ModLink slug="l--fluidCols" /> | カラム幅に応じて自動折り返す段組みレイアウト |
| <ModLink slug="l--switchCols" /> | 複数列と 1 列を自動切り替え |


## Atomic Modules

レイアウトの最小単位となるモジュール群です。

| クラス | 概要 |
|---|---|
| <ModLink slug="a--decorator" /> | 装飾用の空要素 |
| <ModLink slug="a--divider" /> | コンテンツ間の区切り線 |
| <ModLink slug="a--icon" /> | SVG アイコンを出力 |
| <ModLink slug="a--spacer" /> | 要素間にスペースを確保する空要素 |
