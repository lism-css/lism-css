Lismでは、次のようなモジュールクラスを使います。

- `is--{module}`: 複数の種類のモジュールに対してつけ外しできるような汎用性の高い状態モジュール群。
- `l--{module}`: レイアウトの構成単位となるようなモジュール群。
- `a--{module}`: レイアウトの最小単位となるモジュール群。
- `c--{module}`: その他のより具体的な役割を持つコンポーネント群。（他のクラスとの組み合わせを前提にした設計になっているものが多い。）


基本的に、State Modulesを除いた各レイヤーの他モジュールクラスとの複数併用はできません。  
たとえば、`.l--`と`.c--`は併用できますが、`.l--XXX`と`.l--YYY`、`.c--XXX`と`.c--YYY`は同一属性内では併用しないものとします。


## State Modules

要素に対して汎用的につけ外しできる状態モジュール群です。

| クラス | 概要 |
|---|---|
| [`is--container`](/docs/modules/is--container/) | コンテナクエリの基準要素を定義する |
| [`is--wrapper`](/docs/modules/is--wrapper/) | コンテンツ幅を制御するラッパー要素 |
| [`is--layer`](/docs/modules/is--layer/) | 親要素に重ねるオーバーレイレイヤー |
| [`is--linkBox`](/docs/modules/is--linkBox/) | ボックス全体をリンク化する |
| [`is--vertical`](/docs/modules/is--vertical/) | 縦書きモードにする |
| `is--side` | `l--sideMain` の side 要素用（→ [`l--sidemain`](/docs/modules/l--sidemain/) 参照） |
| `is--skipFlow` | `l--flow` 内で次の兄弟との余白を打ち消す（→ [`l--flow`](/docs/modules/l--flow/) 参照） |


## Layout Modules

レイアウトの構成単位となるモジュール群です。

| クラス | 概要 |
|---|---|
| [`l--box`](/docs/modules/l--box/) | コンテンツをグループ化するシンプルなモジュール |
| [`l--flow`](/docs/modules/l--flow/) | 子要素間の余白を自動管理する flow-root モジュール |
| [`l--frame`](/docs/modules/l--frame/) | 直下のメディア要素を自身のサイズに合わせて表示 |
| [`l--flex`](/docs/modules/l--flex/) | Flexbox でコンテンツを配置 |
| [`l--cluster`](/docs/modules/l--cluster/) | 横方向に並べて自動折り返し |
| [`l--stack`](/docs/modules/l--stack/) | 要素を縦方向に配置 |
| [`l--sidemain`](/docs/modules/l--sidemain/) | サイドとメインの 2 カラムレイアウト |
| [`l--grid`](/docs/modules/l--grid/) | CSS Grid でコンテンツを配置 |
| [`l--tilegrid`](/docs/modules/l--tilegrid/) | 均等な行列グリッド（タイル型レイアウト） |
| [`l--center`](/docs/modules/l--center/) | 上下左右中央揃えで配置 |
| [`l--columns`](/docs/modules/l--columns/) | ブレイクポイントごとに列数を指定するカラムレイアウト |
| [`l--fluidcols`](/docs/modules/l--fluidcols/) | カラム幅に応じて自動折り返す段組みレイアウト |
| [`l--switchcols`](/docs/modules/l--switchcols/) | 複数列と 1 列を自動切り替え |


## Atomic Modules

レイアウトの最小単位となるモジュール群です。

| クラス | 概要 |
|---|---|
| [`a--decorator`](/docs/modules/a--decorator/) | 装飾用の空要素 |
| [`a--divider`](/docs/modules/a--divider/) | コンテンツ間の区切り線 |
| [`a--icon`](/docs/modules/a--icon/) | SVG アイコンを出力 |
| [`a--spacer`](/docs/modules/a--spacer/) | 要素間にスペースを確保する空要素 |
