import { SwitchColumns, Box } from 'lism-css/astro';


`l--switchColumns` は、**複数列以上↔︎1列への切り替えをコンテナクエリやメディアクエリを使わずに実装する**クラスです。


## CSS

<SrcCode path="src/scss/primitives/layout/_switchColumns.scss" />


## Lismコンポーネント

### Import

<ImportPackage component="SwitchColumns" />

### 専用Props

| プロパティ | 説明 |
|---|---|
| `breakSize` <PropBadge type="cssvar">`--breakSize`</PropBadge> | 横並びを維持するのに必要な幅（親のサイズ）を指定できます。[szトークン](/docs/tokens/#size)の指定ができます。|


## Usage

### デフォルト幅で利用する

  <PreviewCode slot="tab" label="JSX">
