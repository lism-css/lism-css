import { Divider, Flex, Box, Stack } from 'lism-css/astro';

`a--divider`は、コンテンツ間に区切り線を描画するためのモジュールです。`--bdc`は`var(--divider)`に初期セットされます。


## CSS

<SrcCode path="src/scss/modules/atomic/_divider.scss" />


## Lismコンポーネント

`<Divider>`は、デフォルトで`aria-hidden="true"`が付与されます。

### Import

<ImportPackage component="Divider" />

### 出力されるHTML構造

```pug
div.a--divider
```

### 専用Props

| プロパティ | 説明 |
|---|---|
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |
