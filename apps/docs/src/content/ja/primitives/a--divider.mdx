import { Divider, Flex, Box, Stack } from 'lism-css/astro';

`a--divider`は、コンテンツ間に区切り線を描画するためのクラスです。`--bdc`は`var(--divider)`に初期セットされます。


## CSS

<SrcCode path="src/scss/primitives/atomic/_divider.scss" />


## Lismコンポーネント

`<Lism atomic="divider" aria-hidden="true" />`のエイリアスとして、`<Divider>`を用意しています。


### Import

<ImportPackage component="Divider" />

### 出力されるHTML構造

```pug
div.a--divider[aria-hidden="true"]
```


## Usage

### 基本例
