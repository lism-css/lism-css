import { Spacer, Box, Flex } from 'lism-css/astro';


`a--spacer`は、要素間のスペースを確保するためのクラスです。


## CSS

<SrcCode path="src/scss/primitives/atomic/_spacer.scss" />


## Lismコンポーネント

`<Spacer>`は、デフォルトで`aria-hidden="true"`が付与されます。

### Import

<ImportPackage component="Spacer" />

### 出力されるHTML構造

```pug
div.a--spacer
```

### 専用Props

:::info
`<Spacer>`では、`w`と`h`に対してSPACEトークンの値が使えます。
:::

:::info
`<Lism atomic="spacer" />` とすることで`<Spacer />`と等価の出力を得られます。
:::
