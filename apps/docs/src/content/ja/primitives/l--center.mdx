import { Center, Text } from 'lism-css/astro';
import { DummyText } from '@lism-css/ui/astro';

`l--center` は、**要素を上下左右中央揃えで配置する**ためのクラスです。


## CSS

<SrcCode path="src/scss/primitives/layout/_center.scss" />


## Lismコンポーネント

### Import

<ImportPackage component="Center" />


## Usage

### 水平方向に中央配置

Centerが**高さを持たない**場合は、コンテンツを水平方向に対して中央揃えで配置するだけとなります。


<Preview>
  <PreviewTitle>
    高さの設定が特にない時
  </PreviewTitle>
  <PreviewArea resize p="15">
    <Center bd p="30">
      <Text fz="l">TEXT</Text>
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <Center bd p="30">
      <Text fz="l">TEXT</Text>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html
    <div class="l--center -bd -p:30">
      <p class="-fz:l">TEXT</p>
    </div>
    ```
  </PreviewCode>
</Preview>

<Preview>
  <PreviewTitle>
    長いテキストの改行は左寄せになります。テキストの読みやすさを損ないません。
  </PreviewTitle>
  <PreviewArea resize p="15">
    <Center bd p="30" g="15">
      <Text fz="l">TEXT</Text>
      <DummyText length="xs" lang="ja" />
      <DummyText lang="ja" />
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <Center bd p="30" g="15">
      <Text fz="l">TEXT</Text>
      <p>ロレム・イプサムの座り雨。</p>
      <p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。Elitも穏やかに続いていきますが、積み重ねられてきた「LiberroyとFoogの取り組み」は、余白のようなものです。</p>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html
    <div class="l--center -bd -p:30 -g:15">
      <p class="-fz:l">TEXT</p>
      <p>ロレム・イプサムの座り雨。</p>
      <p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。Elitも穏やかに続いていきますが、積み重ねられてきた「LiberroyとFoogの取り組み」は、余白のようなものです。</p>
    </div>
    ```
  </PreviewCode>
</Preview>


### 上下左右中央に配置する

Centerに**高さやアスペクト比**が設定されている時、垂直方向に対しても中央揃えになります。

<Preview>
  <PreviewTitle>
    上下左右中央に配置
  </PreviewTitle>
  <PreviewArea p="15">
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">ロレム・イプサムの座り雨。</Text>
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx 'ar="3/2"'
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">ロレム・イプサムの座り雨。</Text>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html "-ar:3/2"
    <div class="l--center -bd -g:10 -ar:3/2">
      <p class="-fz:l">TEXT</p>
      <p class="-fz:s">ロレム・イプサムの座り雨。</p>
    </div>
    ```
  </PreviewCode>
</Preview>
