Lism CSSを利用するための手順を解説します。


大きく分けて2通りの導入方法ができます。

## CSS設計のみを導入したい場合

シンプルなHTMLサイトの場合など、CSSファイルのみを読み込む場合はCDN経由で利用できます。


<PreviewTitle>CDN経由でスタイルシートを読み込む</PreviewTitle>
```html
<link href="https://cdn.jsdelivr.net/npm/lism-css@0.12.0/dist/css/main.css" rel="stylesheet" />
```

もちろん、CSSファイルをダウンロードして読み込んでもOKです。


## Lism専用のJSXコンポーネントを使用したい場合

React(.tsx)、Astro(.astro)形式のコンポーネントを `lism-css`としてnpmパッケージで配布しています。

### 1. パッケージ(`lism-css`)のインストール

<Preview>
  <PreviewTitle>パッケージのインポート</PreviewTitle>
  <PreviewCode slot='tab' label='npm'>
  ```bash frame="none"
  npm i lism-css
  ```
  </PreviewCode>
  <PreviewCode slot='tab' label='pnpm'>
  ```bash frame="none"
  pnpm add lism-css
  ```
  </PreviewCode>
  <PreviewCode slot='tab' label='yarn'>
  ```bash frame="none"
  yarn add lism-css
  ```
  </PreviewCode>
</Preview>




### 2. CSS(`main.css`)の読み込み

<PreviewTitle>グローバルスタイルとして読み込んでください。</PreviewTitle>
```js
import 'lism-css/main.css';
```

例：Next.jsであれば、`_app.js`や`layout.js`でimportしてください。



### 3. コンポーネントの読み込み

<Preview>
<PreviewTitle>各コンポーネントのインポート</PreviewTitle>
<PreviewCode slot='tab' label='.jsx'>
```jsx
import { Box, Flex, ... } from 'lism-css/react';
```
</PreviewCode>
<PreviewCode slot='tab' label='.astro'>
```jsx
import { Box, Flex, ... } from 'lism-css/astro';
```
</PreviewCode>
</Preview>



:::warning
::title[.astroを読み込む場合の注意点]

`.astro`ファイルをnode_modeules内から直接読み込むことができず、追加の設定が必要な場合があります。  
"**Directory import '...' is not supported resolving ES modules** ..." というエラーが発生した場合、`astro.config.js`の`vite.ssr.noExternal`に`"lism-css"`を追加してください。

<PreviewTitle>`astro.config.js`</PreviewTitle>
```js
import { defineConfig } from 'astro/config';

export default defineConfig({
  // ...other settings
  vite: {
    ssr: {
      noExternal: ['lism-css'],
    },
  },
});
```

 
<small>（パッケージ配布方法の改善で対応できるかどうか調整中です...）</small>
:::



## Lism UI の導入

アコーディオンやタブ、モーダルなどの**UIコンポーネント**は、別パッケージ `@lism-css/ui` として提供しています。

### パッケージとして利用する

<Preview>
  <PreviewCode slot='tab' label='npm'>
  ```bash frame="none"
  npm i @lism-css/ui
  ```
  </PreviewCode>
  <PreviewCode slot='tab' label='pnpm'>
  ```bash frame="none"
  pnpm add @lism-css/ui
  ```
  </PreviewCode>
</Preview>


```js frame="none"
// Reactコンポーネントをインポートする例
import { Accordion, Tabs, Button } from '@lism-css/ui/react';

// .astro でインポートする例
import { Accordion, Tabs, Button } from '@lism-css/ui/astro';
```


### CLI でプロジェクトにコピーする

`@lism-css/cli` を使うと、UIコンポーネントのソースコードを自分のプロジェクトにコピーし、自由にカスタマイズできます。

```bash frame="none"
# コンポーネントを追加（初回は対話形式でセットアップ）
npx @lism-css/cli add accordion

# 複数コンポーネントをまとめて追加
npx @lism-css/cli add accordion modal tabs

# 全コンポーネントを追加
npx @lism-css/cli add --all
```
