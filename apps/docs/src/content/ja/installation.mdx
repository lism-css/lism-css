Lism CSSを利用するための手順を解説します。


## CSS設計のみを導入する

シンプルなHTMLサイトの場合など、CSSファイルのみを読み込む場合はCDN経由で利用できます。


<PreviewTitle>CDN経由でスタイルシートを読み込む</PreviewTitle>
```html
<link href="https://cdn.jsdelivr.net/npm/lism-css@0.14.0/dist/css/main.css" rel="stylesheet" />
```

もちろん、CSSファイルをダウンロードして読み込んでもOKです。


## クイックスタート

ゼロからセットアップする場合は `create lism` を使うのが最短です。テンプレートと出力先は対話プロンプトで選択できます。

<Preview>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm create lism@latest
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm create lism@latest
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn create lism
  ```
  </PreviewCode>
</Preview>

`@lism-css/cli` の `lism create` と等価です。


### テンプレートを指定して実行する

`--template` オプションを付けると、対話プロンプトをスキップしてテンプレートを直接指定できます。

<Preview>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm create lism@latest --template astro-minimal
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm create lism@latest -- --template astro-minimal
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn create lism --template astro-minimal
  ```
  </PreviewCode>
</Preview>


### 利用可能なテンプレート

| テンプレート | 説明 |
|---|---|
| `astro-minimal` | Astro ベースの最小構成 |


## React/Astro コンポーネントを使用する

React(.tsx)、Astro(.astro)形式のコンポーネントを `lism-css`としてnpmパッケージで配布しています。

### 1. パッケージ(`lism-css`)のインストール

<Preview>
  <PreviewTitle>パッケージのインポート</PreviewTitle>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm i lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm add lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn add lism-css
  ```
  </PreviewCode>
</Preview>




### 2. CSS(`main.css`)の読み込み

<PreviewTitle>グローバルスタイルとして読み込んでください。</PreviewTitle>
```js
import 'lism-css/main.css';
```

例：Next.jsであれば、`_app.js`や`layout.js`でimportしてください。



### 3. コンポーネントの読み込み

<Preview>
<PreviewTitle>各コンポーネントのインポート</PreviewTitle>
<PreviewCode slot="tab" label=".jsx">
```jsx
import { Box, Flex, ... } from 'lism-css/react';
```
</PreviewCode>
<PreviewCode slot="tab" label=".astro">
```jsx
import { Box, Flex, ... } from 'lism-css/astro';
```
</PreviewCode>
</Preview>



## Lism UI の導入

アコーディオンやタブ、モーダルなどの**UIコンポーネント**は、別パッケージ `@lism-css/ui` として提供しています。

### パッケージとして利用する

<Preview>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm i @lism-css/ui
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm add @lism-css/ui
  ```
  </PreviewCode>
</Preview>


```js frame="none"
// Reactコンポーネントをインポートする例
import { Accordion, Tabs, Button } from '@lism-css/ui/react';

// .astro でインポートする例
import { Accordion, Tabs, Button } from '@lism-css/ui/astro';
```


### CLI でプロジェクトにコピーする

`@lism-css/cli` の `lism ui add` を使うと、UIコンポーネントのソースコードを自分のプロジェクトにコピーし、自由にカスタマイズできます。コンポーネント名は `import` するときと同じ PascalCase で指定します。

```bash frame="none"
# コンポーネントを追加（初回は対話形式でセットアップ）
npx @lism-css/cli ui add Accordion

# 複数コンポーネントをまとめて追加
npx @lism-css/cli ui add Accordion Modal Tabs NavMenu

# 全コンポーネントを追加
npx @lism-css/cli ui add --all
```

名前は大文字小文字・ハイフン・アンダースコアを無視して解決するため、`NavMenu` / `navmenu` / `nav-menu` / `nav_menu` のいずれでも同じコンポーネントに解決されます。
