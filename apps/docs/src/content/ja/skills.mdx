プロジェクトに SKILL.md ファイルを配置することで、AIエージェントに Lism CSS の設計ルール・Props・トークン・命名規則などを伝えることができます。

### ファイル構成

スキルは以下の4ファイルで構成されています。

| ファイル | 内容 |
|---------|------|
| `SKILL.md` | エントリポイント — 概要・パッケージ構成・実装ルール・参照リスト |
| `props.md` | Props システム — 主要Props一覧・Prop Class記法・is--/set-- Props・レスポンシブ対応 |
| `tokens.md` | デザイントークン — 余白・フォントサイズ・角丸・影・カラー・パレット |
| `css-rules.md` | CSS設計ルール — Layer構造・命名規則・プレフィックス・カスタムCSS追加ルール |

GitHub: [lism-css/lism-css/.claude/skills/lism-css-best-practices/](https://github.com/lism-css/lism-css/tree/main/.claude/skills/lism-css-best-practices)

</Tabs.Root>
