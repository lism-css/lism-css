プロジェクトに SKILL.md ファイルを配置することで、AIエージェントに Lism CSS の設計ルール・Props・トークン・命名規則などを伝えることができます。


GitHub: [lism-css/skills/lism-css-guide/](https://github.com/lism-css/lism-css/tree/main/skills/lism-css-guide)


## セットアップ

`@lism-css/cli` の `lism skill add` を使うと、使用中の AI ツールを自動検出して SKILL.md を配置できます。

<Preview>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npx @lism-css/cli skill add
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm dlx @lism-css/cli skill add
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn dlx @lism-css/cli skill add
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="bun">
  ```bash frame="none"
  bunx @lism-css/cli skill add
  ```
  </PreviewCode>
</Preview>

対話モードでは、プロジェクト直下のマーカー（`.claude/`, `.cursor/` 等）を検出して候補をチェック済みで表示します。非対話でツールを明示指定する場合は、以下のようなフラグを渡します。

```bash frame="none"
# 単一ツールに配置
npx @lism-css/cli skill add --claude

# 複数ツールを同時に
npx @lism-css/cli skill add --claude --cursor

# 全ツール
npx @lism-css/cli skill add --all
```

### 配置先の対応表

| フラグ | 配置先 |
|--------|--------|
| `--claude` | `.claude/skills/lism-css-guide` |
| `--codex` | `.agents/skills/lism-css-guide` |

### アップデート

バージョン差分を確認してから更新できます。

```bash frame="none"
# 同梱スキルとローカルの version を比較
npx @lism-css/cli skill check

# 最新版で上書き
npx @lism-css/cli skill update --claude
```

最新情報が気になる場合は [GitHub リポジトリ](https://github.com/lism-css/lism-css/tree/main/skills/lism-css-guide) を直接チェックしてください。


### [skills.sh](https://skills.sh) 経由での配置

[skills.sh](https://skills.sh) をお使いの場合は、`skills add lism-css/lism-css` でも同じスキルを取得できます。

```bash frame="none"
npx skills add lism-css/lism-css
```


### ファイル構成

スキルは以下の10ファイルで構成されています。

| ファイル | 内容 |
|---------|------|
| `SKILL.md` | エントリポイント — 概要・パッケージ構成・実装ルール・参照リスト |
| `tokens.md` | デザイントークン — 余白・フォントサイズ・角丸・影・カラー・パレット |
| `base-styles.md` | ベーススタイリング — Reset CSS・HTML要素のベーススタイル・CSS変数概要 |
| `set-class.md` | `set--` クラス — `set--plain`/`set--shadow`/`set--hov`/`set--transition` 等 |
| `primitive-class.md` | Primitive クラス — is--/l--/a-- と Component（c--）クラスの一覧と用途 |
| `utility-class.md` | ユーティリティクラス — u-- クラスの一覧 |
| `property-class.md` | Property Class — `-{prop}:{value}` 記法・主要Prop一覧 |
| `prop-responsive.md` | レスポンシブ対応 — ブレークポイント・コンテナクエリ・指定方法 |
| `components.md` | コンポーネントシステム — コア・レイアウト・セマンティック・UI コンポーネント一覧 |
| `css-rules.md` | CSS設計ルール — Layer構造・命名規則・プレフィックス・カスタムCSS追加ルール |
