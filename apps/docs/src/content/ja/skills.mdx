プロジェクトに SKILL.md ファイルを配置することで、AIエージェントに Lism CSS の設計ルール・Props・トークン・命名規則などを伝えることができます。


GitHub: [lism-css/skills/lism-css-guide/](https://github.com/lism-css/lism-css/tree/main/skills/lism-css-guide)


## セットアップ

### skills.sh 経由

[skills.sh](https://skills.sh) を使うと、以下のコマンドで簡単にスキルを取得できます。

<Preview>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npx skills add lism-css/lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm dlx skills add lism-css/lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn dlx skills add lism-css/lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="bun">
  ```bash frame="none"
  bunx skills add lism-css/lism-css
  ```
  </PreviewCode>
</Preview>


### gh コマンド経由

[GitHub CLI](https://cli.github.com/) を使っている場合は、`gh skill install` でも取得できます。

```bash frame="none"
gh skill install lism-css/lism-css lism-css-guide
```


### lism-cli 経由

`lism-cli` の `lism skill add` からも同じスキルを取得できます。プロジェクト直下への配置のみ対応していますが、`skill check` コマンドで同梱スキルとローカルの差分を確認できるのが特徴です。

<Preview>

対話モードでは、プロジェクト直下のマーカー（`.claude/`, `.cursor/` 等）を検出して候補をチェック済みで表示します。非対話でツールを明示指定する場合は、以下のようなフラグを渡します。

```bash frame="none"
# 単一ツールに配置
npx lism-cli skill add --claude

# 複数ツールを同時に
npx lism-cli skill add --claude --cursor

# 全ツール
npx lism-cli skill add --all
```

#### 配置先の対応表

| フラグ | 配置先 |
|--------|--------|
| `--claude` | `.claude/skills/lism-css-guide` |
| `--codex` | `.agents/skills/lism-css-guide` |
| `--cursor` | `.cursor/skills/lism-css-guide` |
| `--windsurf` | `.windsurf/skills/lism-css-guide` |
| `--cline` | `.cline/skills/lism-css-guide` |
| `--copilot` | `.github/skills/lism-css-guide` |
| `--gemini` | `.gemini/skills/lism-css-guide` |
| `--junie` | `.junie/skills/lism-css-guide` |


#### アップデート

同梱スキルとローカルのファイル差分を確認してから更新できます。

```bash frame="none"
# 同梱スキルとローカルのファイル差分をチェック（変更 / 追加 / 削除のサマリ表示）
npx lism-cli skill check

# 最新版で上書き
npx lism-cli skill update --claude
```

`skill check` は配置済みの各ファイルを sha256 で比較し、変更 / 追加 / 削除されたファイルをまとめて表示します。最新情報が気になる場合は [GitHub リポジトリ](https://github.com/lism-css/lism-css/tree/main/skills/lism-css-guide) を直接チェックしてください。
