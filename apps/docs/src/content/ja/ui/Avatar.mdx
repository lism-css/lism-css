## How to use

`@lism-css/ui` パッケージで`Avatar`(`.c--avatar`)として提供しています。

### Import

<ImportPackage component='Avatar' packageName='@lism-css/ui' />


### Props

| プロパティ | 説明 |
|---|---|
|`size`| アバターのサイズを指定します。|
|`src`| アバター画像のパス |
|`alt`| altテキスト |


### 基本構造

<Preview>
  <PreviewCode slot='tab' label='React / Astro'>
```jsx
<Avatar src="" alt="" />
```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
```html
<div class="c--avatar l--frame -ar:1/1 -w -bdrs:99" style="--w: 1.5em">
  <img src="" alt="" width="100%" height="100%" decoding="async" />
</div>
```
  </PreviewCode>
</Preview>


{/* |`mediaProps`| 上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。| */}
