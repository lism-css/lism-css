## How to use

`@lism-css/ui` パッケージで`Alert`として提供しています。


### Import
<ImportPackage component="Alert" packageName="@lism-css/ui" />


### Props

| プロパティ | 説明 |
|---|---|
|`type`|`keycolor`と`icon`の組み合わせを呼び出すキーワードを指定できます。初期値:`"alert"`|
|`icon`| [`Icon`](/docs/primitives/a--icon/)コンポーネントに渡すアイコンを指定できます。|
|`keycolor`| キーカラーを指定します。|
|`flow`|コンテンツを囲む`Flow`要素の`flow`プロパティに渡されます|
|`layout`|レイアウトプリミティブを切り替えます。初期値:`"flex"`|
{/* | `iconProps` | アイコン（`<Icon>`）に渡すpropsを指定できます。| */}
{/* - `variant`: `'fill'` | `'outline'` ? */}



### ソースコード

ソースコードは [GitHub](https://github.com/lism-css/lism-css/tree/main/packages/lism-ui/src/components/Alert) で公開しています。
