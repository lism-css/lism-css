## How to use

`@lism-css/ui` パッケージで`Badge`として提供しています。


### Import
<ImportPackage component='Badge' packageName='@lism-css/ui' css='style.css' />


### 基本構造

<Preview>
	<PreviewCode slot='tab' label='React, Astro'>
		```jsx
		<Badge>...</Badge>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html
		<span class='c--badge -d:in-flex -fz:xs -lh:xs -py:5 -px:10 -bdrs:10'>...</span>
		```
	</PreviewCode>
</Preview>

### Props

| プロパティ | 説明 |
|---|---|
|`variant`<small>(必須)</small>| `c--badge--{variant}` クラスを出力します。`outline`を用意しています。|
