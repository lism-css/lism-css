## How to use

`@lism-css/ui` パッケージで`Button`(`c--button`)として提供しています。


### Import

<ImportPackage component="Button" packageName="@lism-css/ui" css="style.css" />


### Props

| プロパティ | 説明 |
|---|---|
|`variant`| `c--button--{variant}` クラスを出力します。`fill`,`outline`の2種類のスタイルをこの例では用意しています。|
{/* |`layout`| Buttonを構成するレイアウト用コンポーネントを指定できます。デフォルトは`Lism`です。| */}
{/* |`icon`, `endIcon` | ボタンテキストの左右に表示するアイコンを指定できます。| */}
{/* |`hover`| デフォルトは`op`が指定されています。| */}


### ソースコード

ソースコードは [GitHub](https://github.com/lism-css/lism-css/tree/main/packages/lism-ui/src/components/Button) で公開しています。

## Opt-in

### `hov="reverse"`

ホバー時に、fill,outlineが入れ替わるような [`-hov`クラス](/docs/property-class/hov)を追加する例を紹介します。

<Preview css={`
.-hov\\:reverse {
  --hov-c: var(--bgc);
  --hov-bgc: var(--c);
}
.-hov\\:reverse:where(.c--button--fill) {
  --hov-bgc: transparent;
}
.-hov\\:reverse:where(.c--button--outline) {
  --hov-c: var(--base);
}
@media (any-hover: hover) {
  .-hov\\:reverse:hover {
    background-color: var(--hov-bgc) !important;
    color: var(--hov-c) !important;
  }
}`}>
  <PreviewTitle>例</PreviewTitle>
  <PreviewArea p="20" bg=":stripe" layout="">

      </Button>
    </Flex>
  </PreviewArea>

  <PreviewCode slot="tab" label="追加CSS">

    &:where(.c--button--outline) {

    }
