ダミーのプレースホルダーテキストを出力します。複数の言語とテキスト長に対応しています。


## How to use

`@lism-css/ui` パッケージで`DummyText`として提供しています。


### Import

```jsx
import { DummyText } from '@lism-css/ui/astro';
// or
import { DummyText } from '@lism-css/ui/react';
```


### Props

| プロパティ | 説明 |
|---|---|
| `lang` | 言語を指定できます。デフォルトで`en`。そのほか、`ja`, `ar`をサポートしています。 |
| `length` | ダミーテキストの長さを指定できます。有効な値は `xs`, `s`, `m`, `l`, `xl`, `codes` です。デフォルトは`m`です。 |
| `pre` | ダミーテキストの前に表示するテキストを指定できます。 |
| `as` | HTMLタグを指定できます。デフォルトは`p`です。 |
| `offset` | ダミーテキストのオフセットを指定できます。`,`、`.`単位で区切って切り捨てします。（長さは短くなります） |
