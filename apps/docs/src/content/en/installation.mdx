This guide explains how to set up and use Lism CSS.


## CSS-only setup

For simple HTML sites or other cases where you only need the CSS file, you can use it via CDN.


<PreviewTitle>Load via CDN</PreviewTitle>
```html
<link href="https://cdn.jsdelivr.net/npm/lism-css@0.14.0/dist/css/main.css" rel="stylesheet" />
```

You can also download the CSS file and load it locally.


## Quick start

The fastest way to start from scratch is using `create lism`. The template and output directory can be chosen through interactive prompts.

<Preview>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm create lism@latest
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm create lism@latest
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn create lism
  ```
  </PreviewCode>
</Preview>

This is equivalent to `lism create` from `@lism-css/cli`.


### Specify a template

Pass `--template` to skip the interactive prompt and choose a template directly.

<Preview>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm create lism@latest --template astro-minimal
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm create lism@latest -- --template astro-minimal
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn create lism --template astro-minimal
  ```
  </PreviewCode>
</Preview>


### Available templates

| Template | Description |
|---|---|
| `astro-minimal` | Minimal Astro-based setup |


## Use React/Astro components

React (.jsx) and Astro (.astro) components are distributed as an npm package.

### 1. Install the package

<Preview>
  <PreviewTitle>Install the package</PreviewTitle>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm i lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm add lism-css
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="yarn">
  ```bash frame="none"
  yarn add lism-css
  ```
  </PreviewCode>
</Preview>




### 2. Import the CSS

<PreviewTitle>Import it as a global style.</PreviewTitle>
```js
import 'lism-css/main.css';
```

For example, in Next.js, import it in `_app.js` or `layout.js`.



### 3. Import components

<Preview>
<PreviewTitle>Import individual components</PreviewTitle>
<PreviewCode slot="tab" label=".jsx">
```jsx

```
</PreviewCode>
<PreviewCode slot="tab" label=".astro">
```jsx

```
</PreviewCode>
</Preview>



## Installing Lism UI

**UI components** such as accordions, tabs, and modals are provided as a separate package, `@lism-css/ui`.

### Use as a package

<Preview>
  <PreviewCode slot="tab" label="npm">
  ```bash frame="none"
  npm i @lism-css/ui
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="pnpm">
  ```bash frame="none"
  pnpm add @lism-css/ui
  ```
  </PreviewCode>
</Preview>


```js frame="none"
// Example: importing React components
import { Accordion, Tabs, Button } from '@lism-css/ui/react';

// Example: importing in .astro
import { Accordion, Tabs, Button } from '@lism-css/ui/astro';
```


### Copy to your project with the CLI

Using `@lism-css/cli`'s `lism ui add`, you can copy UI component source code directly into your project and customize it freely. Component names are specified in PascalCase — the same form you use when importing.

```bash frame="none"
# Add a component (interactive setup on first run)
npx @lism-css/cli ui add Accordion

# Add multiple components at once
npx @lism-css/cli ui add Accordion Modal Tabs NavMenu

# Add all components
npx @lism-css/cli ui add --all
```

Case, hyphens, and underscores are ignored when resolving names, so `NavMenu` / `navmenu` / `nav-menu` / `nav_menu` all resolve to the same component.

Set up AI coding tools (Claude Code, Cursor, Codex, etc.) to work effectively with Lism CSS.
