Lism CSS provides design tokens for major CSS properties.

- [**Color Tokens**](/docs/tokens/colors/) — Semantic colors and palette colors
- [**Typography Tokens**](/docs/tokens/typography/) — Font family, size, line-height, letter-spacing
- [**Spacing Tokens**](/docs/tokens/spacing/) — Fibonacci-based spacing scale

| Token value | Variable | Class | Default value |
|---|---|---|---|

## `SHADOW`


{/* <Callout type='warning'>
box-shadow は少し特殊で、ユーティリティクラス内で複数のshadow用変数が組み合わされています。
</Callout> */}

| Token value | Variable | Class |
|---|---|---|
|`10`|`--bxsh--10`|`-bxsh:10`|
|`20`|`--bxsh--20`|`-bxsh:20`|
|`30`|`--bxsh--30`|`-bxsh:30`|
|`40`|`--bxsh--40`|`-bxsh:40`|

The value of `--bxsh--{token}` is composed of the following variables.

| Component variable | Default value |
|---|---|
|`--shc`|`hsl(220 4% 8% / 5%)`|
|`--shsz--5`|`0px 1px 3px`|
|`--shsz--10`|`0px 2px 6px`|
|`--shsz--20`|`0px 4px 12px`|
|`--shsz--30`|`0px 8px 24px`|
|`--shsz--40`|`0px 16px 48px`|

## `SIZE`

Tokens for content sizing.

| Token value | Variable |
|---|---|
|`xs`|`--sz--xs`|
|`s`|`--sz--s`|
|`m`|`--sz--m`|
|`l`|`--sz--l`|
|`xl`|`--sz--xl`|
|`min`|`--sz--min`|

Available for use with `max-sz`, [`.is--wrapper`](/docs/module-class/#is--wrapper), and similar utilities.
