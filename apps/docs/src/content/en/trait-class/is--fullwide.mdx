---

`is--fullwide` expands an element to **fill its parent**. By default it sets `max-inline-size: 100%`, and when the parent has `has--gutter`, it adds the gutter amount and **breaks out past the gutter** to reach the parent edges.

## Using with Lism Components

It can be applied via the `isFullwide` property on **any component**.

| Property | Output |
|---|---|
| `isFullwide`| `is--fullwide` |


## Usage

### Full-width content inside `has--gutter`

```html
<div class="has--gutter">
  <p>Regular content inside the gutter</p>
  <img class="is--fullwide" src="..." alt="" />
  <p>Regular content inside the gutter</p>
</div>
```

Use this when you want to place full-width images or banners that ignore the surrounding `has--gutter` spacing.
