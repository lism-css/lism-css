import { DummyText } from '@lism-css/ui/astro';

`is--layer` is a class for creating overlay layers absolutely positioned on top of a parent element.


## CSS

<SrcCode path="src/scss/trait/is/_layer.scss" />


## Using with Lism Components

A `<Layer>` component is available as an alias for `<Lism isLayer>`.

### Import

<ImportPackage component="Layer" />


## Usage

### Example

  <PreviewTitle>Example</PreviewTitle>

### Using backdrop-filter

You can apply `backdrop-filter` via the `style` prop.

<Preview>
  <PreviewTitle>Applying a blur to the background</PreviewTitle>
  <PreviewArea p="15">
    <Box pos="relative" p="40">
      <Text fz="2xl" fw="bold" ta="center">BACKGROUND</Text>
      <Layer style={{ backdropFilter: 'blur(3px)' }} />
    </Box>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx {3}
    <Box pos="relative" p="40">
      <Text fz="2xl" fw="bold" ta="center">
        BACKGROUND
      </Text>
      <Layer style={{ backdropFilter: 'blur(3px)' }} />
    </Box>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html {3}
    <div class="l--box -pos:relative -p:40">
      <p class="-fz:2xl -fw:bold -ta:center">BACKGROUND</p>
      <div class="is--layer" style="backdrop-filter: blur(3px)"></div>
    </div>
    ```
  </PreviewCode>
</Preview>
