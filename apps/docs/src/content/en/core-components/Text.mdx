import { Text, Lism, Dummy } from "lism-css/astro";

`<Text>` is a Lism component that outputs a `<p>` tag.

:::info
`<Text>` is an alias for `<Lism as='p'>`. All [Lism Props](/docs/core-components/Lism/) are available.  
You can change the output tag with the `as` prop: `p`(default) / `div` / `blockquote` / `address` / `figcaption` / `pre`
:::
