Semantic Wrappers are named components that function as aliases for `<Lism as='tag'>`.  
Use them to make the intent of HTML elements explicit in your code.

:::info
All [Lism Props](/docs/core-components/Lism/) are available on every Semantic Wrapper.
:::


## Import

<ImportPackage component='Text, Inline, Group, Heading, Link, List, ListItem' />


## Overview

| Component | Default `as` | Purpose | Available `as` tags |
|---|---|---|---|
| `Text` | `p` | Text content block elements | `div` / `blockquote` / `address` / `figcaption` / `pre` |
| `Inline` | `span` | Phrasing content | `em` / `strong` / `small` / `code` / `time` / `i` / `b` / `mark` / `abbr` / `cite` / `kbd` |
| `Group` | `div` | Semantic structural wrapper | `section` / `article` / `figure` / `nav` / `aside` / `header` / `footer` / `main` / `fieldset` / `hgroup` |
| `Heading` | `h2` | Headings (`level` prop for h1–h6) | — |
| `Link` | `a` | Links | — |
| `List` | `ul` | Lists | `ol` / `dl` |
| `ListItem` | `li` | List items | `dt` / `dd` |

## Inline

`<Inline>` is a Lism component that outputs a `<span>` tag.  
Use it when you want to apply styles to a portion of text.

<Preview>
  <PreviewTitle>`<Inline>` output example</PreviewTitle>
  <PreviewArea p='20'>
    <Text>This is <Inline fw="bold" c="blue">highlighted</Inline> text.</Text>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Text>This is <Inline fw="bold" c="blue">highlighted</Inline> text.</Text>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <p>This is <span class="-fw:bold -c" style="--c:var(--blue)">highlighted</span> text.</p>
    ```
  </PreviewCode>
</Preview>

## Group

`<Group>` is a Lism component that outputs a `<div>` tag.  
Use it as a general-purpose container for grouping elements.

:::note[Difference from `<Lism>`]
`<Lism>` also outputs a `<div>` by default, but using `<Group>` makes the intent — "this is a grouping container" — explicit in your code.
:::

<Preview>
  <PreviewTitle>`<Group>` output example</PreviewTitle>
  <PreviewArea p='20'>
    <Group p="20" bgc="base-2" bdrs="10">
      <Dummy />
    </Group>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Group p="20" bgc="base-2" bdrs="10">
      Lorem ipsum text...
    </Group>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <div class="-p:20 -bgc:base-2 -bdrs:10">
      Lorem ipsum text...
    </div>
    ```
  </PreviewCode>
</Preview>

## Heading

`<Heading>` is a Lism component that outputs heading tags (`<h1>` to `<h6>`).  
Use the `level` prop to specify the heading level.

| Property | Type | Default | Description |
|---|---|---|---|
| `level` | `'1' \| '2' \| '3' \| '4' \| '5' \| '6'` | `'2'` | Specifies the heading level. |

<Preview>
  <PreviewTitle>`<Heading>` output example</PreviewTitle>
  <PreviewArea p='20'>
    <Heading level="2">Heading Text</Heading>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <Heading level="2">Heading Text</Heading>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <h2>Heading Text</h2>
    ```
  </PreviewCode>
</Preview>
