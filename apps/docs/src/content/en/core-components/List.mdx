import { List, ListItem } from "lism-css/astro";

`<List>` outputs a `<ul>` tag, and `<ListItem>` outputs a `<li>` tag.

:::info
`<List>` is an alias for `<Lism as="ul">`, and `<ListItem>` is an alias for `<Lism as="li">`. All [Lism Props](/docs/core-components/Lism/) are available.  
`<List>` `as` prop: `ul`(default) / `ol` / `dl`  
`<ListItem>` `as` prop: `li`(default) / `dt` / `dd`
:::

## Ordered List

Use `as="ol"` to output an `<ol>` tag.

<Preview>
  <PreviewTitle>`<List as="ol">` output example</PreviewTitle>

## Definition List

Use `<List as="dl">` with `<ListItem as="dt">` and `<ListItem as="dd">` to create a definition list.

<Preview>
  <PreviewTitle>`<List as="dl">` output example</PreviewTitle>
  <PreviewArea p='20'>
    <List as="dl">
      <ListItem as="dt" fw="bold">Term 1</ListItem>
      <ListItem as="dd">Description for term 1.</ListItem>
      <ListItem as="dt" fw="bold">Term 2</ListItem>
      <ListItem as="dd">Description for term 2.</ListItem>
    </List>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx
    <List as="dl">
      <ListItem as="dt" fw="bold">
        Term 1
      </ListItem>
      <ListItem as="dd">Description for term 1.</ListItem>
      <ListItem as="dt" fw="bold">
        Term 2
      </ListItem>
      <ListItem as="dd">Description for term 2.</ListItem>
    </List>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html
    <dl>
      <dt class="-fw:bold">Term 1</dt>
      <dd>Description for term 1.</dd>
      <dt class="-fw:bold">Term 2</dt>
      <dd>Description for term 2.</dd>
    </dl>
    ```
  </PreviewCode>
</Preview>
