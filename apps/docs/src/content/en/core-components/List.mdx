import { List, ListItem, Lism, Dummy } from "lism-css/astro";

`<List>` outputs a `<ul>` tag, and `<ListItem>` outputs a `<li>` tag.

:::info
`<List>` is an alias for `<Lism as='ul'>`, and `<ListItem>` is an alias for `<Lism as='li'>`. All [Lism Props](/docs/core-components/Lism/) are available.
:::

## Import

<ImportPackage component='List, ListItem' />
