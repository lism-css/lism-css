import { Box, Flex, Stack, Divider, Spacer } from 'lism-css/astro';

Border-related specifications in Lism follow a slightly unique convention.

- Use the `-bd` class to show a border on all sides.
- To show a border on a single side, use the `.-bd-{side}` class. (Note: only block and inline directions are supported by default.)
- Fine-grained styles for these borders are controlled by three variables: `--bdc`, `--bdw`, and `--bds`.
- To change the border direction at a breakpoint, adjust `--bdw`.

<PreviewTitle>css</PreviewTitle>
```scss
:where(.-bd, [class*=' -bd-'], [class^='-bd-']) {
  --bds: solid;
  --bdw: 1px;
  --bdc: var(--divider);
  border-width: var(--bdw);
  border-color: var(--bdc);
}
.-bd {
  border-style: var(--bds);
}
.-bd-x {
  border-inline-style: var(--bds);
}
.-bd-y {
  border-block-style: var(--bds);
}
.-bd-x-s {
  border-inline-start-style: var(--bds);
}
.-bd-x-e {
  border-inline-end-style: var(--bds);
}
.-bd-y-s {
  border-block-start-style: var(--bds);
}
.-bd-y-e {
  border-block-end-style: var(--bds);
}
.-bd-t {
  border-top-style: var(--bds);
}
.-bd-b {
  border-bottom-style: var(--bds);
}
.-bd-l {
  border-left-style: var(--bds);
}
.-bd-r {
  border-right-style: var(--bds);
}
```


Accordingly, components also accept `bd`, `bd-{side}`, `bdc`, `bdw`, and `bds`, where `bdc`, `bdw`, and `bds` are output as CSS variables.

Additionally, `bdw` supports breakpoint specification.

<Preview>
  <PreviewTitle>Usage example</PreviewTitle>
  <PreviewArea p='30'>
    <Box bd p="10">
      <DummyText length="s" />
    </Box>
    <Box bd bds="dashed" bdw="4px" bdc="red" p="15">
      <DummyText length="s" />
    </Box>
  </PreviewArea>
  <PreviewCode label='JSX' slot='tab'>
    ```jsx
    <Box bd p='10'>
      <p>Lorem ipsum dolor sit amet. Consectetur adipiscing elit, sed do eiusmod tempor Incididunt ut.</p>
    </Box>
    <Box bd bds='dashed' bdw='4px' bdc='red' p='15'>
      <p>Lorem ipsum dolor sit amet. Consectetur adipiscing elit, sed do eiusmod tempor Incididunt ut.</p>
    </Box>
    ```
  </PreviewCode>
  <PreviewCode label='HTML' slot='tab'>
    ```html "-bd" "--bdc" "--bdw" "--bds"
    <div class="l--box -bd -p:10">
      <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor.</p>
    </div>
    <div class="l--box -bd -bds:dashed -p:15" style="--bdw: 4px; --bdc: var(--red)">
      <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor.</p>
    </div>
    ```
  </PreviewCode>
</Preview>



## Side Specification


:::info
To apply a border on only one side, use the `.-bd-{side}` class.
:::

When different styles are needed for each side, you can combine the `.-bd` class with `--bdw`, `--bds`, and `--bdc` as follows:

<Preview>
  <PreviewTitle>Per-side specification of `bdw`, `bds`, `bdc`</PreviewTitle>
  <PreviewArea resize p='30'>
    <Box bd bdw="0 0 1px .5em" bds="dashed solid" bdc="var(--purple) var(--blue)" p="15">
      border
    </Box>
  </PreviewArea>
  <PreviewCode label='JSX' slot='tab'>
    ```jsx
    <Box bd bdw="0 0 1px .5em" bds="dashed solid" bdc="var(--purple) var(--blue)" p="15">
      border
    </Box>;
    ```
  </PreviewCode>
  <PreviewCode label='HTML' slot='tab'>
    ```html
    <div class="l--box -bd -p:15" style="--bdw: 0 0 1px 0.5em; --bds: dashed solid; --bdc: var(--purple) var(--blue)">border</div>
    ```
  </PreviewCode>
</Preview>




## Switching bdw at Breakpoints

`bdw` supports breakpoint specification.

<Preview>
  <PreviewTitle>Example of `bd`</PreviewTitle>
  <PreviewArea p='30' resize>
    <Box bd bdw={['1px', '3px', '6px']} p="15">
      border
    </Box>
  </PreviewArea>
  <PreviewCode label='JSX' slot='tab'>
    ```jsx
    <Box bd bdw={['1px', '3px', '6px']} p="15">
      border
    </Box>;
    ```
  </PreviewCode>
  <PreviewCode label='HTML' slot='tab'>
    ```html
    <div class="l--box -bd -bdw_sm -bdw_md -p:15" style="--bdw: 1px; --bdw_sm: 3px; --bdw_md: 6px">border</div>
    ```
  </PreviewCode>
</Preview>



### Switching the border side

By switching `bdw` at breakpoints, you can also change which side the border appears on.

<Preview>
  <PreviewTitle>Switching the border side</PreviewTitle>
  <PreviewArea p='30' resize>
    <Flex fxd={['column', 'row']}>
      <Box px="15" py="5">
        Box
      </Box>
      <Box px="15" py="5" bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>
        Box
      </Box>
      <Box px="15" py="5" bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>
        Box
      </Box>
    </Flex>
  </PreviewArea>
  <PreviewCode label='JSX' slot='tab'>
    ```jsx
    <Flex fxd={['column', 'row']}>
      <Box px="15" py="5">
        Box
      </Box>
      <Box px="15" py="5" bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>
        Box
      </Box>
      <Box px="15" py="5" bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>
        Box
      </Box>
    </Flex>;
    ```
  </PreviewCode>
  <PreviewCode label='HTML' slot='tab'>
    ```html
    <div class="l--flex -fxd:col -fxd_sm" style="--fxd_sm: row">
      <div class="l--box -px:15 -py:5">Box</div>
      <div class="l--box -px:15 -py:5 -bd-x-s -bd-y-s -bdw_sm" style="--bdw: 1px 0; --bdw_sm: 0 1px">Box</div>
      <div class="l--box -px:15 -py:5 -bd-x-s -bd-y-s -bdw_sm" style="--bdw: 1px 0; --bdw_sm: 0 1px">Box</div>
    </div>
    ```
  </PreviewCode>
</Preview>
