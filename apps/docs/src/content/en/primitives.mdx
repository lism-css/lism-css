---
title: 'Primitive Class'
description: 'An overview of Lism CSS primitive classes (l--, a--) and their usage.'
---


Lism provides **Primitive classes** as small building blocks for composing layouts.

- `l--{name}`: Layout Primitives that serve as the building blocks of layouts.
- `a--{name}`: Atomic Primitives that represent the smallest units of a layout.

By combining these, you can build complex layouts. The `c--` prefixed **[Component Class](/en/docs/css-methodology/#component-class)** represents concrete UI parts composed from these Primitives.

:::info
Primitive classes are standalone classes that do not have variations (such as `--outline`) or child element classes (such as `_header`). Only `c--` Component Classes follow a BEM-like structure.
:::

For `is--*` classes that declare a role on an element (Trait Classes), see [Trait Class](/en/docs/trait-class/).


#### Combination Rules

| Combination | Allowed | Notes |
|---|---|---|
| `l--` × `l--` | NG | Same category cannot be combined |
| `a--` × `a--` | NG | Same category cannot be combined |
| `l--` × `is--` | OK | Multiple Traits can be combined |
| `a--` × `is--` | OK | |
| `l--` × `a--` | Discouraged | Not intended to coexist |
| `l--` × `c--` | OK | e.g. `c--nav l--flex` |
| `a--` × `c--` | OK | |


## Layout Primitives

Primitives that serve as the building blocks of layouts.

| Class | Description |
|---|---|
| <ModLink slug="l--box" /> | Simple primitive for grouping content |
| <ModLink slug="l--flow" /> | Flow-root primitive that auto-manages spacing between children |
| <ModLink slug="l--frame" /> | Fits child media to its own size |
| <ModLink slug="l--flex" /> | Lays out content using Flexbox |
| <ModLink slug="l--cluster" /> | Arranges items horizontally with automatic wrapping |
| <ModLink slug="l--stack" /> | Arranges elements vertically |
| <ModLink slug="l--sideMain" /> | Two-column layout with side and main areas |
| <ModLink slug="l--grid" /> | Lays out content using CSS Grid |
| <ModLink slug="l--tileGrid" /> | Uniform row-column grid (tile layout) |
| <ModLink slug="l--center" /> | Centers content both horizontally and vertically |
| <ModLink slug="l--columns" /> | Column layout with breakpoint-based column counts |
| <ModLink slug="l--fluidCols" /> | Fluid columns that wrap based on column width |
| <ModLink slug="l--switchCols" /> | Automatically switches between multi-column and single-column |


## Atomic Primitives

Primitives that represent the smallest units of a layout.

| Class | Description |
|---|---|
| <ModLink slug="a--decorator" /> | Empty element for decoration |
| <ModLink slug="a--divider" /> | Divider line between content |
| <ModLink slug="a--icon" /> | Outputs an SVG icon |
| <ModLink slug="a--spacer" /> | Empty element for spacing between elements |
