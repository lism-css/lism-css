"**Lism CSS**" is a lightweight CSS design framework for **building the structural skeleton of websites with a smooth, beautiful rhythm**.

- Typography design that naturally brings a comfortable rhythm to your entire site.
- A layout-first module architecture that makes wireframing straightforward.
- A flexible, responsive utility design powered by CSS variables.

It is a unique CSS design system that blends the layout primitives and harmonic modular scaling of [Every Layout](https://every-layout.dev/), the utility-first approach of [Tailwind CSS](https://tailwindcss.com/), and the layer architecture of [ITCSS](https://www.xfive.co/blog/itcss-scalable-maintainable-css-architecture/).


## Why Lism CSS?

- **No need to design your own CSS architecture.** — From base styling and `@layer` hierarchy to tokens and naming conventions, Lism takes full ownership of your site's entire CSS design.
- **Works by simply importing the CSS.** — No build step or configuration required. Just import a single lightweight stylesheet and you're ready to go.
- **Even more comfortable with React & Astro.** — With dedicated components, you can intuitively handle responsive layouts and build structures using props.

## Zero-build, works in any environment.

Lism CSS is fundamentally **driven by CSS design theory**, and a **single CSS file** is its core.

:::check
Since it can be introduced simply by importing the CSS, it works even with **plain HTML sites**.
:::

<PreviewTitle>You can get started in just one line of code, loaded directly from a CDN.</PreviewTitle>
```html
<link href="https://cdn.jsdelivr.net/npm/lism-css@0/dist/css/main.css" rel="stylesheet" />
```

In other words, no special build process is required to use Lism CSS.

That said, using the dedicated **components** for React and Astro included in the `lism-css` npm package will give you an even more comfortable development experience.

You can also customize Lism more flexibly by overriding SCSS settings or introducing a `lism.config.js` file.


## The Lism CSS Ecosystem

Lism CSS provides not only the core CSS library, but also a suite of packages to support your development, all as open source.

| Package | Description |
|---|---|
| `lism-css` | Core CSS + React / Astro layout components |
| `@lism-css/ui` | A collection of UI components including accordions, tabs, and button links |
| `@lism-css/cli` | A CLI tool to copy UI component source code into your project for customization |
| `@lism-css/mcp` | An MCP server for AI coding tools |

All packages are free and open source (MIT).

## Inspirations

Lism CSS draws heavy inspiration from the following projects.

- [Every Layout](https://every-layout.dev/) — The concepts of **layout primitives** and **harmonic modular scaling** are incorporated.
- [ITCSS](https://itcss.io/) — Referenced for layer hierarchy definitions.
- [Tailwind CSS](https://tailwindcss.com/) — Referenced for the design of providing classes for individual CSS properties.
- [Chakra UI](https://chakra-ui.com/), [MUI](https://mui.com/) — Referenced for the component developer experience.
