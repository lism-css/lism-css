import { FluidCols, Box, Frame } from 'lism-css/astro';
import { DummyText } from '@lism-css/ui/astro';


`l--fluidCols` is a **breakpoint-independent multi-column class** where columns automatically wrap before they shrink below the specified minimum width.


## CSS

<SrcCode path="src/scss/primitives/layout/_fluidCols.scss" />


## Lism Component

### Import

<ImportPackage component="FluidCols" />

### Props

| Property | Description |
|---|---|
| `cols` <PropBadge type="cssvar">`--cols`</PropBadge> | Specifies the minimum width each column should maintain.
| `autoFill` <PropBadge type="cssvar">`--autoMode`</PropBadge> | Switches the auto-wrap mode to `auto-fill`.

### Specifying column width with `auto-fill`

<Preview>
  <PreviewTitle>Using `cols=12em` with `auto-fill`</PreviewTitle>
  <PreviewArea resize p="15">
    <FluidCols cols="12em" autoFill g="20" fz="s">
      <Box bxsh="20" p="20">
        <DummyText length="s" />
      </Box>
      <Box bxsh="20" p="20">
        <DummyText offset="2" />
      </Box>
      <Box bxsh="20" p="20">
        <DummyText length="s" />
      </Box>
    </FluidCols>
  </PreviewArea>
  <PreviewCode>
    ```jsx
    <FluidCols cols="12rem" autoFill g="20" fz="s">
      <Box bxsh="20" p="20">
        <p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。</p>
      </Box>
      <Box bxsh="20" p="20">
        <p>それは静かに歩く仮の言葉です。Elitも穏やかに続いていきますが、積み重ねられてきた「LiberroyとFoogの取り組み」は、余白のようなものです。</p>
      </Box>
      <Box bxsh="20" p="20">
    ```
