import { SwitchColumns, Box } from 'lism-css/astro';


`l--switchColumns` is a class that **switches between multi-column and single-column without using container queries or media queries**.


## CSS

<SrcCode path="src/scss/primitives/layout/_switchColumns.scss" />


## Lism Component

### Import

<ImportPackage component="SwitchColumns" />

### Props

| Property | Description |
|---|---|
| `breakSize` <PropBadge type="cssvar">`--breakSize`</PropBadge> | Specifies the parent width required to maintain the horizontal layout. Accepts [sz tokens](/en/docs/tokens/#size). |
