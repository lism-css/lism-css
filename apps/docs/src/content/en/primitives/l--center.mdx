`l--center` is a class for **centering elements both horizontally and vertically**.


## CSS

<SrcCode path="src/scss/primitives/layout/_center.scss" />


## Lism Component

### Import

<ImportPackage component="Center" />


## Usage

### Horizontal centering

When Center has **no height**, it simply aligns its content horizontally centered.


<Preview>
  <PreviewTitle>
    When no height is set
  </PreviewTitle>
  <PreviewArea resize p="15">
    <Center bd p="30">
      <Text fz="l">TEXT</Text>
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <Center bd p="30">
      <Text fz="l">TEXT</Text>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```jsx
    <div class="l--center -bd -p:30">
      <p class="-fz:l">TEXT</p>
    </div>
    ```
  </PreviewCode>
</Preview>

<Preview>
  <PreviewTitle>
    Long text wraps left-aligned, preserving readability.
  </PreviewTitle>
  <PreviewArea resize p="15">
    <Center bd p="30" g="15">
      <Text fz="l">TEXT</Text>
      <DummyText length="xs" />
      <DummyText />
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx
    <Center bd p="30" g="15">
      <Text fz="l">TEXT</Text>
      <p>Lorem ipsum dolor sit amet.</p>
      <p>Lorem ipsum dolor sit amet. Consectetur adipiscing elit, sed do eiusmod tempor Incididunt ut. Labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut.</p>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```jsx
    <div class="l--center -bd -p:30 -g:15">
      <p class="-fz:l">TEXT</p>
      <p>Lorem ipsum dolor sit amet.</p>
      <p>Lorem ipsum dolor sit amet. Consectetur adipiscing elit, sed do eiusmod tempor Incididunt ut. Labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut.</p>
    </div>
    ```
  </PreviewCode>
</Preview>


### Centering both horizontally and vertically

When Center has a **height or aspect ratio** set, content is also centered vertically.

<Preview>
  <PreviewTitle>
    Centered both horizontally and vertically
  </PreviewTitle>
  <PreviewArea p="15">
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">Lorem ipsum dolor sit amet.</Text>
    </Center>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx 'ar="3/2"'
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">Lorem ipsum dolor sit amet.</Text>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```html "-ar:3/2"
    <div class="l--center -bd -g:10 -ar:3/2">
      <p class="-fz:l">TEXT</p>
      <p class="-fz:s">Lorem ipsum dolor sit amet.</p>
    </div>
    ```
  </PreviewCode>
</Preview>
