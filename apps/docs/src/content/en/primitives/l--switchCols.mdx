import { SwitchCols, Box } from 'lism-css/astro';


`l--switchCols` is a class that **switches between multi-column and single-column without using container queries or media queries**.


## CSS

<SrcCode path="src/scss/primitives/layout/_switchCols.scss" />


## Lism Component

### Import

<ImportPackage component="SwitchCols" />

### Props

| Property | Description |
|---|---|
| `breakSize` <PropBadge type="cssvar">`--breakSize`</PropBadge> | Specifies the parent width required to maintain the horizontal layout. Accepts [sz tokens](/en/docs/tokens/#size). |


## Usage
