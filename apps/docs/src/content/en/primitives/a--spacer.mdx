import { Spacer, Box, Flex } from 'lism-css/astro';


`a--spacer` is a class for reserving space between elements.


## CSS

<SrcCode path="src/scss/modules/atomic/_spacer.scss" />


## Lism Component

`<Spacer>` is rendered with `aria-hidden="true"` by default.

### Import

<ImportPackage component="Spacer" />

### HTML Structure

```pug
div.a--spacer
```

### Props

:::info
In `<Spacer>`, SPACE token values can be used for `w` and `h`.
:::


## Usage

### Basic example

<Preview>
  <PreviewTitle>Usage example of `<Spacer>`</PreviewTitle>
  <PreviewArea resize p="15" layout="">
    <p>Content...</p>
    <Spacer h="100px" />
    <p>Content...</p>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
    ```jsx {2}

    <Spacer h="100px" />
    <p>Content...</p>
    ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
    ```jsx
    <p>Content...</p>
    <div class="a--spacer -h" style="--h:100px" aria-hidden="true"></div>
    <p>Content...</p>
    ```
  </PreviewCode>
</Preview>
