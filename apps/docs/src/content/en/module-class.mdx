---
title: 'Module Class'
description: 'An overview of Lism CSS module classes (is--, l--, a--, c--) and their usage.'
---


Lism uses the following module classes.

- `is--{module}`: Versatile state modules that can be applied or removed across multiple types of elements.
- `l--{module}`: Modules that serve as the building blocks of layouts.
- `a--{module}`: Modules that represent the smallest units of a layout.
- `c--{module}`: Components with more specific roles. (Many are designed to be used in combination with other classes.)


In general, multiple module classes from different layers cannot be combined, except for State Modules.
For example, `.l--` and `.c--` can be used together, but `.l--XXX` and `.l--YYY`, or `.c--XXX` and `.c--YYY`, should not be combined within the same element.


## State Modules

Versatile state modules that can be applied or removed across multiple types of elements.

| Class | Description |
|---|---|
| [`is--container`](/docs/modules/is--container/) | Defines the reference element for container queries |
| [`is--wrapper`](/docs/modules/is--wrapper/) | Wrapper element that controls content width |
| [`is--layer`](/docs/modules/is--layer/) | Overlay layer positioned over a parent element |
| [`is--linkBox`](/docs/modules/is--linkBox/) | Makes the entire box a clickable link |
| [`is--vertical`](/docs/modules/is--vertical/) | Applies vertical writing mode |
| `is--side` | For the side element of `l--sideMain` (→ see [`l--sidemain`](/docs/modules/l--sidemain/)) |
| `is--skipFlow` | Cancels spacing with the next sibling inside `l--flow` (→ see [`l--flow`](/docs/modules/l--flow/)) |


## Layout Modules

Modules that serve as the building blocks of layouts.

| Class | Description |
|---|---|
| [`l--box`](/docs/modules/l--box/) | Simple module for grouping content |
| [`l--flow`](/docs/modules/l--flow/) | Flow-root module that auto-manages spacing between children |
| [`l--frame`](/docs/modules/l--frame/) | Fits child media to its own size |
| [`l--flex`](/docs/modules/l--flex/) | Lays out content using Flexbox |
| [`l--cluster`](/docs/modules/l--cluster/) | Arranges items horizontally with automatic wrapping |
| [`l--stack`](/docs/modules/l--stack/) | Arranges elements vertically |
| [`l--sidemain`](/docs/modules/l--sidemain/) | Two-column layout with side and main areas |
| [`l--grid`](/docs/modules/l--grid/) | Lays out content using CSS Grid |
| [`l--tilegrid`](/docs/modules/l--tilegrid/) | Uniform row-column grid (tile layout) |
| [`l--center`](/docs/modules/l--center/) | Centers content both horizontally and vertically |
| [`l--columns`](/docs/modules/l--columns/) | Column layout with breakpoint-based column counts |
| [`l--fluidcols`](/docs/modules/l--fluidcols/) | Fluid columns that wrap based on column width |
| [`l--switchcols`](/docs/modules/l--switchcols/) | Automatically switches between multi-column and single-column |


## Atomic Modules

Modules that represent the smallest units of a layout.

| Class | Description |
|---|---|
| [`a--decorator`](/docs/modules/a--decorator/) | Empty element for decoration |
| [`a--divider`](/docs/modules/a--divider/) | Divider line between content |
| [`a--icon`](/docs/modules/a--icon/) | Outputs an SVG icon |
| [`a--spacer`](/docs/modules/a--spacer/) | Empty element for spacing between elements |
