Creates a two-column layout that switches between horizontal and vertical arrangements based on the desired content width, without relying on media queries or container queries.

This is useful for layouts such as a main content area with a side panel.

## Structure

```pug
div.l--sideMain
	div
	div.is--side
```

## CSS

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/modules/layout/_sideMain.scss' />


## Import

<ImportPackage component='SideMain' />

## Props

| Property | Output | Description |
|---|---|---|
| `sideW`| <PropBadge type="cssvar">`--sideW`</PropBadge> | Width of the side element |
| `mainW` | <PropBadge type="cssvar">`--mainW`</PropBadge> | Minimum width to maintain for the main element |

{/* In addition, [Props available on `<Flex>`](/components/flex#props) can be used. */}

## Usage

A child element with `.is--side` is required.
The side element's width is fixed to the value set by `--sideW`. When the main element's width falls below `--mainW`, the layout automatically switches to a single column.

#### Reversed order

Using the `-fxd:row-r` class (`flex-direction:row-reverse`) together allows you to reverse only the horizontal order.

:::check
This places the side element on the left when horizontal, and at the bottom when stacked vertically.
:::

				Side Content

## Building a similar layout with `Grid`

`SideMain` (`.l--sideMain`) is not dependent on breakpoints, but you can also build a similar layout using [`Grid`](/components/grid) where the switch point is controlled by a breakpoint.

<Preview>
	<PreviewTitle>Building a SideMain-like layout using `Grid`</PreviewTitle>
	<PreviewArea resize p='20'>
		<Grid gta={[`'.' 'side'`, `'. side'`]} gtc={[null, '1fr 10rem']} g='20'>
			<Box>
				<Dummy />
			</Box>
			<Box ga='side' bgc='blue:20%'>
				Side Content
			</Box>
		</Grid>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "gta=" "gtc="
		<Grid gta={[`'.' 'side'`, `'. side'`]} gtc={[null, '1fr 10rem']} g='20'>
			<Box>
				<Dummy />
			</Box>
			<Box ga='side'>Side Content</Box>
		</Grid>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html "-gd -gd_sm" "'. side' / 1fr 10rem" "-ga" "--ga:side"
		<div class="l--grid -gta -gta_sm -gtc_sm -g:20" style="--gta:'.' 'side';--gta_sm:'. side';--gtc_sm:1fr 10rem">
			<div class='l--box'>
				<p>
					Lorem ipsum dolor sit, amet consectetur adipisicing elit, sed do eiusmod tempor.
					Non facere laudantium ex eos doloribus aut dolore nisi.
				</p>
			</div>
			<div class='l--box -ga' style='--ga:side'>
				Side Content
			</div>
		</div>
		```
	</PreviewCode>
</Preview>
