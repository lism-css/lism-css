`l--center` is a layout module for **centering elements both horizontally and vertically**.

## Usage

### Horizontal centering

When Center has **no height**, it simply aligns its content horizontally centered.

{/* <Callout type="info">
For horizontal centering only, `<Stack ai="center">` also works.
</Callout> */}

{/* For horizontal alignment, this achieves what is known as "intrinsic" centering — if the text is short, it is centered; if it is long, it behaves like normal left-aligned text. */}

<Preview>
  <PreviewTitle>
    When no height is set
  </PreviewTitle>
  <PreviewArea resize p='15'>

      <Text fz="l">TEXT</Text>

  <PreviewCode slot='tab' label='JSX'>

### Centering both horizontally and vertically

When Center has a **height or aspect ratio** set, content is also centered vertically.

<Preview>
  <PreviewTitle>
    Centered both horizontally and vertically
  </PreviewTitle>
  <PreviewArea p='15'>
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">Lorem ipsum dolor sit amet.</Text>
    </Center>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx 'ar="3/2"'
    <Center g="10" ar="3/2" bd>
      <Text fz="l">TEXT</Text>
      <Text fz="s">Lorem ipsum dolor sit amet.</Text>
    </Center>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```html "-ar:3/2"
    <div class="l--center -bd -g:10 -ar:3/2">
      <p class="-fz:l">TEXT</p>
      <p class="-fz:s">ロレム・イプサムの座り雨。</p>
    </div>
    ```
  </PreviewCode>
</Preview>
