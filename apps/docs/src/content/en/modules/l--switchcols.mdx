import { SwitchCols, Box, Frame, Dummy } from 'lism-css/astro';


A layout that **switches between multi-column and single-column without using media queries**.


## Props

| Property | Description |
|---|---|
| `breakSize` <PropBadge type="cssvar">`--breakSize`</PropBadge> | Specifies the parent width required to maintain the horizontal layout. Accepts [sz tokens](/docs/tokens/#size). |
