import { Lism, Box, Frame, Media, Columns, Grid, Divider, Heading } from 'lism-css/astro';
import { DummyText, DummyImage } from '@lism-css/ui/astro';

Creates a column layout that displays content in the specified number of columns at each breakpoint.

## Props

| Property | Description | Default |
|---|---|---|
| `cols` <PropBadge type="cssvar">`--cols`</PropBadge> | Specifies the number of columns. Breakpoint-based values are also supported. | `2` |

{/* - `cols`: Specifies the number of columns. Breakpoint-based values are also supported. */}
You can also use any [props available on `<Grid>`](/docs/modules/l--grid#props).
