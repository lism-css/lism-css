import { Grid, Lism, Box, Layer, Center, Decorator, Frame, Media } from 'lism-css/astro';
import { DummyText } from '@lism-css/ui/astro';


A component for arranging content in a **Grid layout**.


## Import

<ImportPackage component='Grid' />


## Props

|Prop|CSS|
|---|---|
|`gt`|`grid-template`|
|`gta`|`grid-template-areas`|
|`gtc`|`grid-template-columns`|
|`gtr`|`grid-template-rows`|
|`gaf`|`grid-auto-flow`|
|`gar`|`grid-auto-rows`|
|`gac`|`grid-auto-columns`|

### Overlapping child elements

Here is an example of stacking elements using `grid-area:1/1` (`.-ga:1`) without using `position:absolute`.

<Preview>
  <PreviewTitle>Overlapping Grid child elements</PreviewTitle>
  <PreviewArea p='15'>
    <Grid ar="16/9">
      <Frame ga="1/1" w="100%" h="100%" pos="relative">
        <Media src="https://cdn.lism-css.com/img/a-1.jpg" alt="" inferSize />
        <Layer style={{ backdropFilter: 'blur(10px)' }} bgc="rgb(0 0 0 / 40%)" />
      </Frame>
      <Center ga="1/1" fz="3xl" z="1" c="white">SAMPLE TEXT</Center>
    </Grid>
  </PreviewArea>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx 'ga="1/1"'
    <Grid ar="16/9">
      <Frame ga="1/1" w="100%" h="100%" pos="relative">
        <Media src="https://cdn.lism-css.com/img/a-1.jpg" alt="" inferSize />
        <Layer style={{ backdropFilter: 'blur(10px)' }} bgc="rgb(0 0 0 / 40%)" />
      </Frame>
      <Center ga="1/1" fz="3xl" z="1" c="white">
        SAMPLE TEXT
      </Center>
    </Grid>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='HTML'>
    ```jsx "-ga:1"

        <div class="is--layer -bgc" style="backdrop-filter:blur(10px);--bgc:rgb(0 0 0 / 40%)"></div>

        SAMPLE TEXT
      </div>
    </div>
    ```
