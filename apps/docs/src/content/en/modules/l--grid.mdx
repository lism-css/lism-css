import { Grid, Lism, Box, Layer, Center, Decorator, Frame, Media, Dummy } from 'lism-css/astro';


A component for arranging content in a **Grid layout**.


## Import

<ImportPackage component='Grid' />


## Props

|Prop|CSS|
|---|---|
|`gt`|`grid-template`|
|`gta`|`grid-template-areas`|
|`gtc`|`grid-template-columns`|
|`gtr`|`grid-template-rows`|
|`gaf`|`grid-auto-flow`|
|`gar`|`grid-auto-rows`|
|`gac`|`grid-auto-columns`|

### `-gt:repeat`

A `gt` utility class that accepts `cols` for the number of columns and `rows` for the number of rows.

Both support breakpoint-specific values and default to `1`.


<Preview>
	<PreviewTitle>Example</PreviewTitle>
	<PreviewArea resize p='15'>
		<Grid gt='repeat' cols='3' rows='3' g='5' min-h='50svh'>
			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
			<Box p='20' bgc='base-2' gc='span 2'>item5</Box>
			<Center gc='3' gr='1 / -1' p='20' bgc='blue'>item6</Center>
		</Grid>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx " gc=" " gr="
		<Grid gt='repeat' cols='3' rows='3' g='5' min-h='50svh'>
			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
			<Box p='20' bgc='base-2' gc='span 2'>item5</Box>
			<Center gc='3' gr='1 / -1' p='20' bgc='blue'>item6</Center>
		</Grid>
		```
	</PreviewCode>
</Preview>


`cols` and `rows` each support breakpoint-specific values.

<Preview>
	<PreviewTitle>Example</PreviewTitle>
	<PreviewArea resize p='15'>
		<Grid gt='repeat' cols={['2', '3']} rows='4' min-h='50svh' gaf='row dense'>
			<Box gc={['1 / -1','span 2']} gr={['span 2', '1 / -1']} p='20' bgc='base-2'>A</Box>
			<Box p='20' bgc='green:20%'>B</Box>
			<Box p='20' bgc='blue:20%'>C</Box>
			<Box p='20' bgc='red:20%'>D</Box>
			<Box p='20' bgc='purple:20%'>E</Box>
		</Grid>
	</PreviewArea>
	<PreviewCode>
		```jsx "gt='repeat'"

			<Box gc={['1 / -1','span 2']} gr={['span 2', '1 / -1']} p='20' bgc='base-2'>A</Box>

			<Box p='20' bgc='blue:20%'>C</Box>
			<Box p='20' bgc='red:20%'>D</Box>
			<Box p='20' bgc='purple:20%'>E</Box>
		</Grid>
		```
	</PreviewCode>
</Preview>
