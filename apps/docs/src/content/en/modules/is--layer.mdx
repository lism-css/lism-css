import { Media, Layer, Stack, Flex, HTML, Box, Columns, Center, Frame, Dummy } from 'lism-css/astro';



A component that outputs [`.is--layer`](/docs/css-methodology//#is--layer) by default (an alias for `<Lism isLayer>`).



{/* <Reference>
If you simply want to position an element with `position:absolute`, you can also use the `-pos:abs` class.
See: [Positioning with absolute](/components/props/others/absolute)
</Reference> */}


## Props

| Property | Description |
|---|---|
| `blur`,`contrast`,`brightness`,`dropShadow`,`grayscale`,`hueRotate`,`invert`,`saturate`,`sepia` | These values are combined and passed to `style.backdropFilter`. |

### Using backdrop-filter

The `<Layer>` component includes additional support for specifying individual `backdrop-filter` properties — such as `blur` — separately.

<Preview>
	<PreviewTitle>Applying a blur to the background</PreviewTitle>
	<PreviewArea p='15'>
		<Box pos='rel' p='40'>
			<HTML.p fz='2xl' fw='bold' ta='center'>BACKGROUND</HTML.p>
			<Layer blur='3px' />
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx {3}
		<Box pos='rel' p='40'>
			<HTML.p fz='2xl' fw='bold' ta='center'>BACKGROUND</HTML.p>
			<Layer blur='3px' />
		</Box>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html {3}
		<div class="l--box -pos:rel -p:40">
			<p class="-fz:2xl -fw:bold -ta:center">BACKGROUND</p>
			<div class="is--layer" style="backdrop-filter:blur(3px)"></div>
		</div>
		```
	</PreviewCode>
</Preview>

### Media Layer

Here is a layout example using `.is--layer` to place an image as a background behind content.

<Preview>
	<PreviewTitle>Example</PreviewTitle>
	<PreviewArea p='15'>
		<Box pos='rel' py='50' px='40'>
			<Frame isLayer>
				<Media src="https://cdn.lism-css.com/img/a-2.jpg" alt="" width='960' height='640' loading='lazy' decoding='async' />
			</Frame>
			<Layer bgc='black:50%' />
			<Stack pos='rel' g='30' c='white'>
				<Dummy length='s' />
				<Dummy length='l' />
			</Stack>
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
