import { TileGrid, Box, Center } from 'lism-css/astro';


A component for creating **uniform tile-style grid layouts** by specifying `cols` (number of columns) and `rows` (number of rows).

Internally, it uses `grid-template: repeat(rows, 1fr) / repeat(cols, 1fr)`.

## Lism Component

### Import

<ImportPackage component='TileGrid' />

### Props

| Property | Description | Default |
|---|---|---|
| `cols` <PropBadge type="cssvar">`--cols`</PropBadge> | Specifies the number of columns. Breakpoint-specific values are supported. | `1` |
| `rows` <PropBadge type="cssvar">`--rows`</PropBadge> | Specifies the number of rows. Breakpoint-specific values are supported. | `1` |

Other [props available on `<Grid>`](/en/docs/modules/l--grid#props) can also be used.

### Breakpoint-specific values

Both `cols` and `rows` support breakpoint-specific values.

<Preview>
  <PreviewTitle>Example</PreviewTitle>
  <PreviewArea resize p='15'>

      <Box gc={['1 / -1', 'span 2']} gr={['span 2', '1 / -1']} p="20" bgc="base-2">A</Box>
      <Box p="20" bgc="green:20%">B</Box>
      <Box p="20" bgc="blue:20%">C</Box>
      <Box p="20" bgc="red:20%">D</Box>
