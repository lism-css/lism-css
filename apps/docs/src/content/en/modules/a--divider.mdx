import { Divider, Flex, Box, Stack } from 'lism-css/astro';

A component that renders a dividing line. `--bdc` is initialized to `var(--divider)`.


## Structure
```pug
div.a--divider
```


## CSS

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/modules/atomic/_divider.scss' />


## Props

| Property | Description |
|---|---|
| `bdw`,`bds`,`bdc` | Border properties for the dividing line |


## Import

<ImportPackage component='Divider' />
