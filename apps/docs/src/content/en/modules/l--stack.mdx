### Center-aligned layout using Stack

The combination of `<Stack ai="center">` (`.l--stack.-ai:center`) enables what Every Layout calls **"intrinsic centering"** — content is centered only when it is short enough.

<Preview>
  <PreviewTitle>Intrinsic centering</PreviewTitle>
  <PreviewArea resize>
    <Stack ai="center" g="20" p="30" className="has--itemGuide">
      <h3 role="presentation">Heading</h3>
      <Dummy length="s" />
      <Dummy length="l" />
    </Stack>
  </PreviewArea>
  <PreviewCode slot="tab" label="JSX">
  ```jsx "ai='center'"
  <Stack ai="center" g="20" p="30">
    <h3>Heading</h3>
    <p>...</p>
    <p>...</p>
  </Stack>;
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="HTML">
  ```jsx "-ai:center"

    <h3>Heading</h3>
    <p>...</p>
    <p>...</p>
  </div>;
  ```
  </PreviewCode>
</Preview>
