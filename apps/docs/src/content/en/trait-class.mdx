**Trait Classes** declare a role on an element or attach a feature to it.  
They belong to `@layer lism-trait`.

- `is--{name}`: declares a **role / existence** (“is a …”).
- `has--{name}`: attaches a **feature** (“has a …”).


## `is--*` (Role declaration)

| Class | Description |
|---|---|
| <ModLink slug="is--container" /> | Defines the reference element for container queries |
| <ModLink slug="is--wrapper" /> | Wrapper element that controls content width |
| <ModLink slug="is--layer" /> | Overlay layer positioned over a parent element |
| <ModLink slug="is--boxLink" /> | Makes the entire box a clickable link |
| `is--coverLink` | A link with a clickable area covering the entire parent element (→ see <ModLink slug="is--boxLink" />) |
| `is--side` | For the side element of `l--sideMain` (→ see <ModLink slug="l--sideMain" />) |
| `is--skipFlow` | Cancels spacing with the next sibling inside `l--flow` (→ see <ModLink slug="l--flow" />) |

### `has--gutter`

A class that adds predefined horizontal spacing to the left and right sides of content.
It is provided as a separate class from padding utilities to reserve a consistent horizontal margin across the entire site.

| Class name | Spacing | Default value |
|---|---|---|
| `has--gutter` | `--gutter-size` | `var(--s30)` |

### `has--mask`

Applies a mask to the element itself in combination with the `--maskImg` variable.

<SrcCode path="src/scss/trait/_hasMask.scss" />

| Variable | Purpose | Default |
|---|---|---|
| `--maskImg` | Mask image (SVG, etc.) | — |
| `--maskPos` | Mask position | `50%` |
| `--maskSize` | Mask size | `contain` |
| `--maskRepeat` | Mask repeat | `no-repeat` |

In Lism components, use `hasMask` together with `css={{ '--maskImg': getSvgUrl(svgString) }}`.
