import { Lism, Box, Media, Flex, Stack, Spacer, Frame, Text } from 'lism-css/astro';
import { Button } from '@lism-css/ui/astro';


Utility classes defined in the `@lism-base` layer are provided in the format `set--{className}`. They are used to override base styles, redefine tokens, and set initial values for functional properties.

A class for resetting primary styles.

### `set--shadow`

Elements with the `.set--shadow` class have the `--bxsh--` variables that compose the shadow redefined.

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/base/tokens/_shadow.scss' />

By overriding the `--shc` variable here, you can adjust the color tint of the shadow.

<Preview>
  <PreviewTitle>Example of overriding shadow color</PreviewTitle>
  <PreviewArea p='30' nogrid>
    <Stack g="10">
      <Text> Default: </Text>
      <Flex className="set--shadow -g:30" px-s="15">
        <Box bxsh="20" p="30" bdrs="20">
          Box
        </Box>
        <Box bxsh="40" p="30" bdrs="20">
          Box
        </Box>
      </Flex>
      <Spacer h="40" />
      <Text> Overriding variables with set--shadow: </Text>
      <Flex className="set--shadow -g:30" px-s="15" style={{ '--shc': 'hsl(200 50% 50% / 20%)' }}>
        <Box bxsh="20" p="30" bdrs="20">
          Box
        </Box>
        <Box bxsh="40" p="30" bdrs="20">
          Box
        </Box>
      </Flex>
    </Stack>
    <div class="a--spacer" />
  </PreviewArea>
    <PreviewCode slot='tab' label='HTML'>
    ```html "set--shadow"
    <div class="set--shadow l--flex -g:30" style="--shc: hsl(200 50% 50% / 20%)">
      <div class="l--box -bxsh:20 -p:30 -bdrs:20">Box</div>
      <div class="l--box -bxsh:40 -p:30 -bdrs:20">Box</div>
    </div>
    ```
  </PreviewCode>
  <PreviewCode slot='tab' label='JSX'>
    ```jsx "set--shadow"
    <Flex className="set--shadow" g="30" style={{ '--shc': 'hsl(200 50% 50% / 20%)' }}>

        Box
      </Box>
      <Box bxsh="40" p="30" bdrs="20">
        Box
      </Box>
    </Flex>;
    ```
  </PreviewCode>
</Preview>


### `set--bp`

A utility class for applying styles only at or above a specific breakpoint.
It can be used as a workaround for properties that do not support [responsive breakpoints](/docs/responsive/).

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/base/set/_bp.scss' />

<Preview>
  <PreviewTitle>Example: changing text color to red at sm size and above</PreviewTitle>
  <PreviewArea resize>
    <Box className="set--bp" p="20" style={{ color: 'var(--_is_sm) red;' }}>
      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.</p>
    </Box>
  </PreviewArea>
  <PreviewCode slot='tab' label='HTML'>
  ```html "set--bp" "var(--_is_sm) red"
  <div class="set--bp l--box -p:20" style="color: var(--_is_sm) red">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.</p>
  </div>
  ```
  </PreviewCode>
  <PreviewCode slot='tab' label='JSX'>

### `set--cqUnit`

A class for scaling using `cqw` units.

Recalculates font-size `clamp()` values using `cqw` instead of `vw`, and redefines spacing and half-leading token units accordingly.

### `set--innerRs`

A utility class that calculates the inner border radius (`--bdrs--inner`) of a child element based on the parent element's padding and border radius.
The calculation is based on the parent element's (`set--innerRs`) `p` (`--p`) and `bdrs` (`--bdrs`) values. (Both must be specified.)

To apply the calculated `--bdrs--inner` to a child element, use `-bdrs:inner` on that child.

<Preview>
  <PreviewTitle>Usage example</PreviewTitle>
  <PreviewArea p='30'>
    <Lism className="set--innerRs" p="15" bd bxsh="20" bdrs="50">
      <Frame bdrs="inner" ar="og">
        <img src="https://cdn.lism-css.com/img/u-5.jpg" width="1920" height="1280" alt="" />
      </Frame>
    </Lism>
  </PreviewArea>
  <PreviewCode label="JSX" slot="tab">
```jsx
<Lism className="set--innerRs" p="15" bd bxsh="20" bdrs="50">
  <Frame bdrs="inner" ar="og">
    <img src="https://cdn.lism-css.com/img/u-5.jpg" width="1920" height="1280" alt="" />
  </Frame>
</Lism>;
```
  </PreviewCode>
  <PreviewCode label="HTML" slot="tab">
```html
<div class="set--innerRs -p:15 -bd -bxsh:20 -bdrs:50">
  <div class="l--frame -bdrs:inner -ar:og">
    <img src="https://cdn.lism-css.com/img/u-5.jpg" width="1920" height="1280" alt="" />
  </div>
</div>
```
  </PreviewCode>
</Preview>

### `set--gutter`

A class that adds predefined horizontal spacing to the left and right sides of content.
It is provided as a separate class from padding utilities to reserve a consistent horizontal margin across the entire site.

| Class name | Spacing | Default value |
|---|---|---|
| `set--gutter` | `--gutter-size` | `1.5rem` |
