A component for creating navigation menu lists.


## How to use

Provided as `NavMenu` (`.c--navMenu`) from the `@lism-css/ui` package.


### Import

<ImportPackage component='NavMenu' packageName='@lism-css/ui' css='style.css' />


### Props

| Property | Description |
|---|---|
| <PropBadge>`<NavMenu.Root>`</PropBadge><br/>`hovBgc` | Outputs `--hov-bgc`. (For use when `.-hov:bgc` is applied to `.c--navMenu_link`.) |
| <PropBadge>`<NavMenu.Root>`</PropBadge><br/>`hovC` | Outputs `--hov-c`. (For use when `.-hov:c` is applied to `.c--navMenu_link`.) |
| <PropBadge>`<NavMenu.Link>`</PropBadge><br/>`hov` | Specifies the hover style. |

### Adjusting padding for all items at once

Specifying `itemP` sets the `--_item-p` variable, allowing you to control the padding of all items in one place.

However, if you need to change the value at a breakpoint, specify `p` on each `<NavMenu.Link>` individually.

<Preview>
  <PreviewTitle>Specifying `itemP`</PreviewTitle>
  <PreviewArea p='30'>
    <NavMenu.Root bd itemP="30">
      <NavMenu.Item hov="bgc">
        <NavMenu.Link href="#menu1">Menu item</NavMenu.Link>
      </NavMenu.Item>
      <NavMenu.Item hov="bgc" bd-t>
        <NavMenu.Link href="#menu2">Menu item</NavMenu.Link>
      </NavMenu.Item>
      <NavMenu.Item hov="bgc" bd-t>
        <NavMenu.Link href="#menu3">Menu item</NavMenu.Link>
      </NavMenu.Item>
    </NavMenu.Root>
  </PreviewArea>
  <PreviewCode>
    ```jsx "bdc='border'" "itemP='30'"
    <NavMenu.Root bd itemP="30">
      <NavMenu.Item hov="bgc">
        <NavMenu.Link href="#menu1">Menu item</NavMenu.Link>
      </NavMenu.Item>
      <NavMenu.Item hov="bgc" bd-t>
        <NavMenu.Link href="#menu2">Menu item</NavMenu.Link>
      </NavMenu.Item>
      <NavMenu.Item hov="bgc" bd-t>
        <NavMenu.Link href="#menu3">Menu item</NavMenu.Link>
      </NavMenu.Item>
    </NavMenu.Root>
    ```

### Expanding nested menus with an accordion

Since link padding is managed by `--_item-p`, you can use `var(--_item-p)` inside an accordion to apply the same padding amount.

<Preview>
  <PreviewTitle>Example</PreviewTitle>
  <PreviewArea p='20'>
    <NavMenu.Root bd-y hovBgc="base-2">
      <NavMenu.Item>
        <NavMenu.Link hov="bgc" href="#menu1">
          <Fragment>Menu item</Fragment>
        </NavMenu.Link>
      </NavMenu.Item>
      <NavMenu.Item>
        <Accordion.Root>
          <Accordion.Item>
            <Accordion.Heading>
              <Accordion.Button hov="bgc" p="var(--_item-p)">
                <Fragment>Menu item (Accordion)</Fragment>
              </Accordion.Button>
            </Accordion.Heading>
            <Accordion.Panel>
              <NavMenu.Nest>
                <NavMenu.Item>
                  <NavMenu.Link hov="bgc" href="#child-menu1">
                    <Fragment>Nested item</Fragment>
                  </NavMenu.Link>
                </NavMenu.Item>
                <NavMenu.Item>
                  <NavMenu.Link hov="bgc" href="#child-menu2">
                    <Fragment>Nested item</Fragment>
                  </NavMenu.Link>
                </NavMenu.Item>
              </NavMenu.Nest>
            </Accordion.Panel>
          </Accordion.Item>
        </Accordion.Root>
      </NavMenu.Item>
    </NavMenu.Root>
  </PreviewArea>
  <PreviewCode>
