## How to use

Provided as `Alert` from the `@lism-css/ui` package.


### Import
<ImportPackage component="Alert" packageName="@lism-css/ui" />


### Props

| Property | Description |
|---|---|
|`type`| Specifies a keyword that sets a combination of `keycolor` and `icon`. Default: `"alert"` |
|`icon`| Specifies the icon to pass to the [`Icon`](/en/docs/primitives/a--icon/) component. |
|`keycolor`| Specifies the key color. |
|`flow`| Passed to the div element wrapping the content. |
|`layout`| Switches the layout primitive. Default: `"flex"` |



### Source Code

The source code is available on [GitHub](https://github.com/lism-css/lism-css/tree/main/packages/lism-ui/src/components/Alert).
