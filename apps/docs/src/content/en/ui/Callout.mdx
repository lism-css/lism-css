### Props

| Property | Description |
|---|---|
|`keycolor`| Specifies the display color. |
|`icon`| Specifies the icon. |
|`type`| Specifies a preset value combining `keycolor` and `icon`. Default: `"note"` |
|`title`| Specifies the title text. |
|`flow`| Passed to `_body`. |
