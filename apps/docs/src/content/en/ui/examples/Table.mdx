A general-purpose Table layout example that supports sticky display for the first row and first column.

Border styles are controlled via CSS variables to enable sticky display behavior.


## Create Component

An example of creating and using a component called `Table` (`.c--table`).

### Sticky display on scroll

The `[data-table-sticky]` attribute accepts the following values:

- `col`: Fixes the first column
- `head`: Fixes the thead
