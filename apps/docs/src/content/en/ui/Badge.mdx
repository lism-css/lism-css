## How to use

Provided as `Badge` (`.c--badge`) from the `@lism-css/ui` package.


### Import
<ImportPackage component='Badge' packageName='@lism-css/ui' css='style.css' />


### Props

| Property | Description |
|---|---|
|`variant`<small>(optional)</small>| Outputs the `c--badge--{variant}` class. An `outline` style is provided by default. |
