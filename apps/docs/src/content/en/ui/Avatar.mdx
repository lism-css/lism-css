## How to use

Provided as `Avatar` (`.c--avatar`) from the `@lism-css/ui` package.

### Import

<ImportPackage component="Avatar" packageName="@lism-css/ui" />


### Props

| Property | Description |
|---|---|
|`size`| Specifies the size of the avatar. |
|`src`| Path to the avatar image. |
|`alt`| Alt text for the image. |


### Structure

<Preview>
  <PreviewCode slot="tab" label="React / Astro">
