## How to use

Provided as `Button` (`.c--button`) from the `@lism-css/ui` package.


### Import

<ImportPackage component="Button" packageName="@lism-css/ui" css="style.css" />


### Props

| Property | Description |
|---|---|
|`variant`<small>(optional)</small>| Outputs the `c--button--{variant}` class. Two styles are available: `fill` and `outline`. |

### Aligning icons to the edge using `Grid`

Specifying `layout='grid'` switches the layout to a grid layout (`.l--grid` class).
This allows you to create a layout where icons are pinned to the edges of the button.

## Opt-in

### `hov="reverse"`

An example of adding a [`-hov` class](/en/docs/props/hov) that swaps the fill and outline styles on hover.

<Preview css={`
.-hov\\:reverse {
  --hov-c: var(--bgc);
  --hov-bgc: var(--c);
}
.-hov\\:reverse:where(.c--button--fill) {
  --hov-bgc: transparent;
}
.-hov\\:reverse:where(.c--button--outline) {
  --hov-c: var(--base);
}
@media (any-hover: hover) {
  .-hov\\:reverse:hover {
    background-color: var(--hov-bgc) !important;
    color: var(--hov-c) !important;
  }
}`}>
  <PreviewTitle>Example</PreviewTitle>
  <PreviewArea p="20" bg=":stripe" layout="">
    <Flex g="15">
      <Button href="###" hov="reverse" variant="fill">
        <Fragment>Button</Fragment>
      </Button>
      <Button href="###" hov="reverse" variant="outline">
        <Fragment>Button</Fragment>
      </Button>
    </Flex>
  </PreviewArea>

  <PreviewCode slot="tab" label="Additional CSS">
  ```scss
  .-hov\:reverse {
    --hov-c: var(--bgc);
    --hov-bgc: var(--c);
    &:where(.c--button--fill) {
      --hov-bgc: transparent;
    }
    &:where(.c--button--outline) {
      --hov-c: var(--base);
    }
    @media (any-hover: hover) {
      &:hover {
        background-color: var(--hov-bgc) !important;
        color: var(--hov-c) !important;
      }
    }
  }
  ```
  </PreviewCode>
  <PreviewCode slot="tab" label="JSX">
  ```jsx 'hov="reverse"'

    <Button href="###" hov="reverse" variant="fill">
      Button
    </Button>

      Button
    </Button>
  </Flex>
  ```
  </PreviewCode>
</Preview>
