Lism provides content size tokens in the form `--sz--{sz}` (where `{sz}` = `xs`, `s`, `m`, `l`, `xl`), and corresponding `-max-sz` classes are available for each. (`-max-sz:{xs~xl}`)


This page introduces the additional special classes beyond those.


## -max-sz:full, -max-sz:container

The following notable classes are also available:

| Class | Description |
|---|---|
| `-max-sz:full` | When inside a `.has--gutter` element, **expands to fill the parent** by adding the gutter spacing on top of `100%`. |
| `-max-sz:container` | Expands to the size of the **nearest container element**. The value is updated for each `.is--container` class. |
