import { HTML, Box, Flow, Flex, Media, Layer, Frame, Center, Columns, Dummy } from 'lism-css/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportPackage, HelpText, Note, EmbedCode } from "~/components";


直下のメディア要素（`img`,`vieo`,`picture`,`iframe`）を `object-fit:cover` で自身のサイズに合わせて表示します。


## CSS

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/layout/_frame.scss' />


## Props

| プロパティ | 説明 |
|---|---|
| `ar` | フレームの比率を指定できます。|
{/* | `objectPosition` | 直下のメディア要素のobject-positionを指定できます。 | */}

<HelpText>(`ar` 自体はどのコンポーネントにも指定できる CSS Props です。)</HelpText>


## Import

<ImportPackage component='Frame' />

### ar はどのコンポーネントにも指定できます

<Note title='arで指定できる値について'>
`ar`(`aspect-ratio`のCSS Props)は他のコンポーネントでも指定できる汎用的なプロパティです。  
[使用できる値についてはこちらをご覧ください。](/css/prop-class/#aspect-ratio)
</Note>
