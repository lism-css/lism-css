import { Grid, Lism, Box, Center, Decorator, Frame, Media, Link, Dummy } from 'lism-css/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportPackage, PropBadge, Callout } from "~/components";


コンテンツを**Gridレイアウトで配置する**ためのコンポーネントです。


## Import

<ImportPackage component='Grid' />


## Props

|Prop|CSS|
|---|---|
|`gd`|`grid`|
|`gt`|`grid-template`|
|`gta`|`grid-template-areas`|
|`gtc`|`grid-template-columns`|
|`gtr`|`grid-template-rows`|
|`gaf`|`grid-auto-flow`|
|`gar`|`grid-auto-rows`|
|`gac`|`grid-auto-columns`|

### 子要素を重ねて表示する

`position:absolute`を使わず、`grid-area:1/1`(`.-ga:1`)を利用して要素を重ねる例を紹介します。

<Preview>
	<PreviewTitle>Gritの子要素を重ねて表示する</PreviewTitle>
	<PreviewArea p='10'>
		<Grid ar='16/9' ji='c' ai='c'>
			<Frame ga='1/1' w='100%' h='100%'>
				<Media src='/img/img-1.jpg' op='mid'/>
			</Frame>
			<Decorator ga='1/1' blur='80px' bgc='white' w='50%' h='4em'/>
			<Box ga='1/1' fz='2xl' z='1'>SAMPLE TEXT</Box>
		</Grid>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "ga='1/1'"
		<Grid ar='16/9' ji='c' ai='c'>
			<Frame ga='1/1' w='100%' h='100%'>
				<Media src='/img/img-1.jpg' op='mid'/>
			</Frame>
			<Decorator ga='1/1' blur='80px' bgc='white' w='50%' h='4em'/>
			<Box ga='1/1' fz='2xl' z='1'>SAMPLE TEXT</Box>
		</Grid>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```jsx "-ga:1"
		<div class="l--grid -ar:16/9 -ji:c -ai:c">
			<div class="l--frame -w:100% -h:100% -ga:1">
				<img class="-op:mid" src="/img/img-1.jpg">  
			</div>
			<div class="l--decorator -bgc -w -h -ga:1" style="--bgc:var(--white);--w:50%;--h:4em;filter:blur(80px)" aria-hidden="true"></div>
			<div class="l--box -fz:2xl -z:1 -ga:1">SAMPLE TEXT</div>
		</div>
		```
	</PreviewCode>
</Preview>

## Variants


### `variant='repeat'`

#### Props

|Prop|CSS|
|---|---|
|`cols`|`--cols`|
|`rows`|`--rows`|


<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea resize p={20}>
		<Grid variant='repeat' cols='3' rows='3' g='5'>
			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
			<Box p='20' bgc='base-2' gc='span 2'>item5</Box>
			<Center gc='3' gr='1 / -1' p='20' bgc='blue'>item6</Center>
		</Grid>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx " gc=" " gr="

			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
		```
