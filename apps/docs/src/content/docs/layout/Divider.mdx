区切り線を描画するコンポーネントです。`--bdc`が`var(--c--line)`に初期セットされます。


## 構造
```pug
div.l--divider
```


## CSS

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/layout/_divider.scss' />


## Props

| プロパティ | 説明 |
|---|---|
| `bdw`,`bds`,`bdc` | 区切り線のボーダープロパティ |


## Import

<ImportPackage component='Divider' />
