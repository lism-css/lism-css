### Prop Class の省略名についての変更

これまではできる限り省略した短いクラス名を採用していましたが、**極端な省略は避ける**ように変更しました。

`-{prop}:{value}`において、
- `{prop}`部分については、主要なプロパティのみ 1~3文字への極端な省略名を用います。
- `{value}`部分については、基本的には省略はせず、長いキーワード値は意味がわかる程度に省略します。

`{value}`部分の省略を避けることで、`{prop}`側が省略されていてもその組み合わせを見てどのプロパティを示すか予測しやすいようになりました。

### 廃止になったもの

- 使用頻度の少ないであろう`grid`プロパティ指定用の`gd`を廃止しました。`gt`, `gar`, `gac` を使用してください。
- `l--flex`, `l--grid`があるので`-d:f`, `-d:g`を廃止しました。
- 使用頻度が少ないと判断し、`-d:i`, `-d:ib` を廃止しました。
- `-fw:100`~`-fw:900` を廃止し、セマンティックトークンを増やしました。
- `-c:mix` 系クラスを廃止しました。`black:50%`のような指定方法は引き続き利用できますが、単純に`color-mix()`で`style`属性に出力されます。
- `l--frame`があるので不要と判断し、`obf` (`object-fit`)をLism Propから除外しました。
- `iso`, `bxsz`, `trf`, `trfo`, `fltr`, `bdfltr` をLism Propから除外しました。
- `u--clipText`を廃止し、`-bgclip:text`と`-c:trsp`を追加しました。
- ov-x,ov-yのhidden,clip
