### Props

| プロパティ | 説明 |
|---|---|
|`variant`<small>(必須)</small>| `c--button-{variant}` クラスを出力します。`fill`,`outline`の2種類のスタイルをこの例では用意しています。|
{/* |`layout`| Buttonを構成するレイアウト用コンポーネントを指定できます。デフォルトは`Lism`です。| */}
{/* |`icon`, `endIcon` | ボタンテキストの左右に表示するアイコンを指定できます。| */}
{/* |`hover`| デフォルトは`fade`が指定されています。| */}


{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}

### `Grid`でアイコンを端に寄せる

`layout={Grid}`を指定すると gridレイアウト（`.l--grid`クラス）に切り替えることができます。  
これにより、アイコンを端に寄せるレイアウトを組むこともできます。

### `hov="reverse"`

ホバー時に、fill,outlineが入れ替わるような [`-hov`クラス](/css/hover)を追加する例を紹介します。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea p='30' bg='u:stripe' isFlow={null}>
		<Flex g='20'>
			<Button href='###' hov='reverse' variant='fill'>Button</Button>
			<Button href='###' hov='reverse' variant='outline'>Button</Button>
		</Flex>
	</PreviewArea>

	<PreviewCode slot='tab' label='追加CSS'>
	```scss
	.-hov\:reverse {
		--hov-c: var(--bgc);
		--hov-bgc: var(--c);
		&:where(.c--button-fill) {
			--hov-bgc: transparent;
		}
		&:where(.c--button-outline) {
			--hov-c: var(--c-base);
		}
		@media (any-hover: hover) {
			&:hover {
				background-color: var(--hov-bgc) !important;
				color: var(--hov-c) !important;
			}
		}
	}
	```
	</PreviewCode>
	<PreviewCode slot='tab' label='JSX'>
