## Create Component

ここからは`Avatar`(`.c--avatar`)というコンポーネントを作って利用する例を紹介していきます。

### Source Code
<ImportSource name='Avatar' react astro />

### Import
<ImportExComponent name='Avatar' react astro />



### Props

| プロパティ | 説明 |
|---|---|
|`size`| アバターのサイズを指定します。|
|`src`| アバター画像のパス |
|`alt`| altテキスト |


{/* |`mediaProps`| 上記以外にメディア側に対してプロパティを指定したい場合はこのプロパティにオブジェクト形式で指定できます。| */}
