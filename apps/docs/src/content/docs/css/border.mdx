Lismでは、border系の指定が少し特殊になっています。

- 全方向表示には `-bd` クラスを使用します。
- 単一方向にボーダーを表示する時は `.-bd-{side}`クラスで指定します。（ただし標準でサポートがあるのは、block, inline形式のみです。）
- これらの細かなスタイルは `--bdc`, `--bdw`, `--bds` の3つの変数で指定します。
- ブレイクポイントでボーダーの方向を変えたい場合、`--bdw` で調整します。

<PreviewTitle>css</PreviewTitle>
```scss
:where(.-bd,[class*='-bd-']) {
	--bds: solid;
	--bdw: 1px;
	--bdc: currentColor;
}
:where(.-bd,[class*='-bd-']) {
	border-width: var(--bdw);
	border-color: var(--bdc);
}
.-bd { border-style: var(--bds) }
.-bd-x { border-inline-style: var(--bds) }
.-bd-y { border-block-style: var(--bds) }
.-bd-s { border-inline-start-style: var(--bds) }
.-bd-e { border-inline-end-style: var(--bds) }
.-bd-bs { border-block-start-style: var(--bds) }
.-bd-be { border-block-end-style: var(--bds) }
```


これに合わせて、コンポーネントでも `bd`, `bd-{side}` `bdc`, `bdw`, `bds` がそれぞれ指定できるようになっており、`bdc`, `bdw`, `bds` はCSS変数が出力されるようになっています。

また、`bdw`はブレイクポイント指定も可能です。

<Preview>
	<PreviewTitle>使用例</PreviewTitle>
	<PreviewArea p='40'>
		<Box bd>
			<Dummy length='s'/>
		</Box>
		<Box bd bds='dashed' bdw='2px' bdc='blue' p='20'>
			<Dummy length='s'/>
		</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd>
			<Dummy length='s'/>
		</Box>
		<Box bd bds='dashed' bdw='2px' bdc='blue' p='20'>
			<Dummy length='s'/>
		</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html "-bd" "-bd\:l" "--bdc" "--bdw" "--bds"
		<div class="l--box -bd">
			<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor. </p>
		</div>
		<div class="l--box -bd -bds:dashed -p:20" style="--bdw:2px;--bdc:var(--blue)">
			<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor. </p>
		</div>
		```
	</PreviewCode>
</Preview>



## 方向指定


<Callout type='info'>
一方向だけのボーダー指定をしたい場合、`.-bd-{side}`クラスを使います。
</Callout>

<Preview>
	<PreviewTitle>block, inline方向での指定</PreviewTitle>
	<PreviewArea p='40' isFlow='50'>
		<Box bd-s p='10' bgc='base-2'>inline-start</Box>
		<Box bd-e p='10' bgc='base-2'>inline-end</Box>
		<Box bd-bs p='10' bgc='base-2'>block-start</Box>
		<Box bd-be bds='dashed' bdw='2px' bdc='red' p='10' bgc='base-2'>block-end</Box>
		<Box bd-x p='10' bgc='base-2'>inline</Box>
		<Box bd-y p='10' bgc='base-2' bdc='blue'>block</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx "bd-s" "bd-e" "bd-bs" "bd-be"
		<Box bd-s p='10' bgc='base-2'>inline-start</Box>
		<Box bd-e p='10' bgc='base-2'>inline-end</Box>
		<Box bd-bs p='10' bgc='base-2'>block-start</Box>
		<Box bd-be bds='dashed' bdw='2px' bdc='red' p='10' bgc='base-2'>block-end</Box>
		<Box bd-x p='10' bgc='base-2'>inline</Box>
		<Box bd-y p='10' bgc='base-2' bdc='blue'>block</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html "-bd-s" "-bd-e" "-bd-bs" "-bd-be"
		<div class="l--box -bd-s -p:10 -bgc:base-2">inline-start</div>
		<div class="l--box -bd-e -p:10 -bgc:base-2">inline-end</div>
		<div class="l--box -bd-bs -p:10 -bgc:base-2">block-start</div>
		<div class="l--box -bd-be -bds:ds -p:10 -bgc:base-2" style="--bdw:2px;--bdc:var(--red)">block-end</div>
		```
	</PreviewCode>
</Preview>

<Spacer h='10' />

複数方向のボーダーそれぞれのスタイルがバラバラの場合、`.-bd`クラスと `--bdw` `--bds` `--bdc` を組み合わせて次のように指定できます。

<Preview>
	<PreviewTitle>`bdw`,`bds`,`bdc`それぞれの方向指定</PreviewTitle>
	<PreviewArea resize p='40'>
		<Box bd bdw='0 0 1px .5em' bds='dashed solid' bdc='var(--purple) var(--blue)' p='20'>
			border
		</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd bdw='0 0 1px .5em' bds='dashed solid' bdc='var(--purple) var(--blue)' p='20'>
			border
		</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html
		<div class="l--box -bd -p:20" style="--bdw:0 0 1px .5em;--bds:dashed solid;--bdc:var(--purple) var(--blue)">
			border
		</div>
		```
	</PreviewCode>
</Preview>




## bdwをブレイクポイントで切り替える

`bdw`は、ブレイクポイント指定に対応しています。

<Preview>
	<PreviewTitle>`bd`の例</PreviewTitle>
	<PreviewArea p='40' resize>
		<Box bd bdw={['1px','3px','6px']} p='20'>border</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd bdw={['1px','3px','6px']} p='20'>border</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html
		<div class="l--box -bd -bdw_sm -bdw_md -p:20" style="--bdw:1px;--bdw_sm:3px;--bdw_md:6px">border</div>
		```
	</PreviewCode>
</Preview>

## 任意の`border`をインライン出力する方法

`bd`に直接値を指定することももちろん可能です。

<Preview>
	<PreviewTitle>`bd`の例</PreviewTitle>
	<PreviewArea p='40' resize>
		<Box bd='solid 1px #555' p='10'>border</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd='solid 1px #555' p='10'>border</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html
		<div class="l--box -p:10" style="border:solid 1px #555">border</div>
		```
	</PreviewCode>
</Preview>
