Lismでは、border系の指定が少し特殊になっています。

- 全方向表示には `-bd` クラスを使用します。
- 単一方向にボーダーを表示する時は `.-bd-{side}`クラスで指定します。（ただし標準でサポートがあるのは、block, inline形式のみです。）
- これらの細かなスタイルは `--bdc`, `--bdw`, `--bds` の3つの変数で指定します。
- ブレイクポイントでボーダーの方向を変えたい場合、`--bdw` で調整します。

<PreviewTitle>css</PreviewTitle>
```scss
:where(.-bd,[class*='-bd-']) {
	--bds: solid;
	--bdw: 1px;
	--bdc: currentColor;
}
:where(.-bd,[class*='-bd-']) {
	border-width: var(--bdw);
	border-color: var(--bdc);
}
.-bd { border-style: var(--bds) }
.-bd-x { border-inline-style: var(--bds) }
.-bd-y { border-block-style: var(--bds) }
.-bd-x-s { border-inline-start-style: var(--bds) }
.-bd-x-e { border-inline-end-style: var(--bds) }
.-bd-y-s { border-block-start-style: var(--bds) }
.-bd-y-e { border-block-end-style: var(--bds) }
```


これに合わせて、コンポーネントでも `bd`, `bd-{side}` `bdc`, `bdw`, `bds` がそれぞれ指定できるようになっており、`bdc`, `bdw`, `bds` はCSS変数が出力されるようになっています。

また、`bdw`はブレイクポイント指定も可能です。

<Preview>
	<PreviewTitle>使用例</PreviewTitle>
	<PreviewArea p='40'>
		<Box bd bdc='line' p='10'>
			<Dummy length='s'/>
		</Box>
		<Box bd bds='dashed' bdw='4px' bdc='red' p='20'>
			<Dummy length='s'/>
		</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd bdc='line' p='10'>
			<Dummy length='s'/>
		</Box>
		<Box bd bds='dashed' bdw='4px' bdc='red' p='20'>
			<Dummy length='s'/>
		</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html "-bd" "--bdc" "--bdw" "--bds"
		<div class="l--box -bd -bdc:line -p:10">
			<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor. </p>
		</div>
		<div class="l--box -bd -bds:dashed -p:20" style="--bdw:4px;--bdc:var(--red)">
			<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor. </p>
		</div>
		```
	</PreviewCode>
</Preview>

一方向だけのボーダー指定をしたい場合、`.-bd-{side}`クラスを使います。

## bdwをブレイクポイントで切り替える

`bdw`は、ブレイクポイント指定に対応しています。

<Preview>
	<PreviewTitle>`bd`の例</PreviewTitle>
	<PreviewArea p='40' resize>
		<Box bd bdw={['1px','3px','6px']} p='20'>border</Box>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Box bd bdw={['1px','3px','6px']} p='20'>border</Box>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html
		<div class="l--box -bd -bdw_sm -bdw_md -p:20" style="--bdw:1px;--bdw_sm:3px;--bdw_md:6px">border</div>
		```
	</PreviewCode>
</Preview>



### ボーダーの方向を切り替える

bdwをブレイクポイントで切り替えることで、ボーダーの方向を切り替えることができます。

<Preview>
	<PreviewTitle>ボーダーの方向を切り替える</PreviewTitle>
	<PreviewArea p='40' resize>
		<Flex fxd={['column','row']}>
			<Box px='20' py='5'>Box</Box>
			<Box px='20' py='5' bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>Box</Box>
			<Box px='20' py='5' bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>Box</Box>
		</Flex>
	</PreviewArea>
	<PreviewCode label='JSX' slot='tab'>
		```jsx
		<Flex fxd={['column','row']}>
			<Box px='20' py='5'>Box</Box>
			<Box px='20' py='5' bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>Box</Box>
			<Box px='20' py='5' bd-x-s bd-y-s bdw={['1px 0', '0 1px']}>Box</Box>
		</Flex>
		```
	</PreviewCode>
	<PreviewCode label='HTML' slot='tab'>
		```html
		<div class='l--flex -fxd:col -fxd_sm' style='--fxd_sm:row'>
			<div class='l--box -px:20 -py:5'>Box</div>
			<div class='l--box -px:20 -py:5 -bd-x-s -bd-y-s -bdw_sm' style='--bdw:1px 0;--bdw_sm:0 1px'>Box</div>
			<div class='l--box -px:20 -py:5 -bd-x-s -bd-y-s -bdw_sm' style='--bdw:1px 0;--bdw_sm:0 1px'>Box</div>
		</div>
		```
	</PreviewCode>
</Preview>
