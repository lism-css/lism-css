## Base Set Utility

`@lism.base`の階層で定義され、トークン再定義や変数のセット、HTMLスタイルの再定義などを行うクラスを `set:{style}` の形式で定義しています。


### `set-revert`

ブラウザ標準スタイル、またはそれに近いスタイルを再定義するためのクラスです。

現状、基本的には ol,ul に対してのみ有効です。

リストタグのベーススタイルについては次のようになっており、ul,liの標準スタイルは reset.css で打ち消した上で、classを持たないかProps Class で始まるリスト、もしくは`set-revert`クラスを指定したリストについては、ブラウザ標準のスタイルへ上書きするような仕様になっています。


<PreviewTitle>CSS</PreviewTitle>
```css
:where(ul, ol) {
	list-style: none;
	padding: 0;
	padding-inline-start: var(--list-px-s, 0);
}

:is(ul, ol):where(:not([class]), [class^='-'], .set-revert) { {
	--list-px-s: 1.5em;
	list-style: revert;
}
```

### `set-plain`

主要スタイルをリセットするためのクラスです。

```css
.set-plain {
	font: inherit;
	width: auto;
	height: auto;
	color: inherit;
	background: none;
	border: none;
	text-decoration: none;
	padding: 0;
}
```

### `set-shadow`

`.set-shadow` クラスを指定した要素では、シャドウを構成する各CSS変数が再定義されます。  

ここで変数を上書きすることで、シャドウの色味などを調整することが可能です。

### `set-hov`

ホバースタイル管理用のCSS変数をセットします。主に子要素のホバースタイルを制御するために使用します。

```scss
@media (any-hover: hover) {
	.set-hov:hover {
		--notHov_: ;
	}
	.set-hov:not(:hover) {
		--isHov_: ;
	}
}
@media (any-hover: none) {
	.set-hov {
		--isHov_: ;
	}
}
```

<Reference>
活用例は [`hover`の解説ページ](/css/hover#set-hov) を参照してください。
</Reference>

### `set-innerRs`

親要素の角丸とPaddingの値から、内側の要素の角丸(`--bdrs--inner`)を計算するユーティリティクラスです。 
親要素(`set--innerRs`)の p(`--p`), bdrs(`--bdrs`) を元に計算します。（これらの指定が必須となります。）

計算された`--bdrs--inner`をセットするには、子要素で`-bdrs:inner`を指定します。

<Preview>
	<PreviewTitle>使用例</PreviewTitle>
	<PreviewArea p='40'>
	<Lism className='set-innerRs' p='20' bd bxsh='20' bdrs='50'>
		<Frame bdrs='inner' ar='og'>
			<img src='/img/img-5.jpg' width='1920' height='1280' alt='' />
		</Frame>
	</Lism>
	</PreviewArea>
	<PreviewCode label="JSX" slot="tab">
```jsx
<Lism className='set-innerRs' p='20' bd bxsh='20' bdrs='50'>

		<img src='/img/img-5.jpg' width='1920' height='1280' alt='' />
```

	</PreviewCode>
