hover時の挙動をコントロールするためのCSS設計について紹介します。


Lismでは、てhover時のスタイルは`-hov:`で始まるPropクラスで定義します。  
また、`has--transition`クラスを併用することでトランジションの設定を行います。


## 基本的な使い方

`.-hov:{prop}` という形式のクラスと、`--hov-{prop}` というcss変数を組み合わせて、hover時の挙動をコントロールします。

## `.has--hover` と特殊変数


<IconBadge /> Lism では、[`has--hover`](/css/layout-state#has--hover)というクラスを定義しており、このクラスでは`--_isHov`変数と`--_notHov`変数を定義しています。

- `--_isHov`は非hover時にのみ空白定義されています。
- `--_notHov`は逆に、hover時にのみ空白定義されています。

```css
@media (any-hover: hover) {
	.-hov\:set:hover {
		--_notHov: ;
	}
	.-hov\:set:not(:hover) {
		--_isHov: ;
	}
}
@media (any-hover: none) {
	.-hov\:set {
		--_isHov: ;
	}
}
```

例えば `var(--_isHov, value)` とすると、`--_isHov`はhover時は未定義となるため、`value`が適用されます。  
非hover時は空白の定義がされているため、無効な値となり無視されます。（ただし同じ要素の同じプロパティに対してすでにスタイルを適用している場合、それも無効となることに注意が必要です。）

以下に例を示します。

<Preview>
	<PreviewTitle>`.-hov:test`を追加</PreviewTitle>
	<PreviewArea p='40'>
		<div className='-p:40 -bd -fz:l has--hover -hov:test'>BOX</div>
	</PreviewArea>

	<PreviewCode slot='tab' label='追加CSS'>
		```scss
		/* 変数は.has--hover で定義済み */
		.-hov\:test {
			color: var(--_isHov, red) var(--_notHov, blue); /* hover時はgreen, 非hover時はblue*/
			background-color: var(--_notHov, lightgray); /* 非hover時はlightgray、hover時は無効な値 */
			box-shadow: var(--_isHov, var(--bxsh-30)); /* hover時はshadowあり、非hover時は無効な値 */
		}
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html "has--hover -hov:test"
		<div className='-p:40 -bd -fz:l has--hover -hov:test'>BOX</div>
		```
	</PreviewCode>
</Preview>


上記の例での `color` では、常に片方が空白でもう片方には値が適用されるような`var()`を2つ並べることでどちらが一方が適用されるというテクニックを使っています。
これにより、hover時と非hover時のスタイルを同列に書くことができます。


### 子要素にホバー状態を渡す

さて、この`has--hover`が便利なのはここからです。
親要素に`has--hover`を付与してその子要素で`--_isHov`変数と`--_notHov`変数をうまく使うことで、hover時の子要素の挙動をコントロールできるようになります。


コアパッケージには、そのような子要素向けのクラスを `-hov:get:xxx` として2つだけ標準で用意しています。

```css

/* 親の has--hover がhoverされたら非表示にする */
.-hov\:get\:hide {
	opacity: var(--_isHov, 0);
}

/* 親の has--hover がhoverされたら表示する */
.-hov\:get\:show {
	opacity: var(--_notHov, 0);
	visibility: var(--_notHov, hidden);
}
```

### `-hov:get`クラスを自作する例


親がhoverされたらズーム表示されるような`-hov:get:zoom`クラスと、テキストが広がるような`-hov:get:letterSpread`クラスを自作して使ってみます。

さらに、親要素で`--transition--duration`を上書きして子要素のトランジション時間を一括で変更してみましょう。

<Preview>
	<PreviewTitle>自作クラスで子要素のホバーアクションを調整する例</PreviewTitle>
	<PreviewArea p='30'>
		<LinkBox
			href='#banner-link'
			hasHover ov='hidden' ar='16/9' bdrs='30'
			style={{'--transition--duration':'.6s'}}
		>
			<Frame isLayer>
				<Media hasTransition hov='get:zoom' src='/img/img-2.jpg' alt='' width='960' height='640' loading='lazy'/>
			</Frame>
			<Layer hasTransition hov='get:hide' grayscale='70%'/>
			<Center isLayer c='#fff' bgc='rgb(0 0 0 / 40%)'>
				<Text hasTransition hov='get:letterSpread' fz='2xl'>LinkBox</Text>
			</Center>
		</LinkBox>
	</PreviewArea>

	<PreviewCode slot='tab' label='追加CSS'>
	```css
	.-hov\:get\:zoom {
		scale: var(--_isHov, 1.1);
	}
	.-hov\:get\:letterSpread {
		letter-spacing: var(--_isHov, 0.125em);
	}
	```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html "has--hover" "-hov:get:zoom" "-hov:get:letterSpread" "--transition--duration"

				<span class="-hov:get:letterSpread -fz:2xl">LinkBox</span>
			</div>
		</a>
		```
