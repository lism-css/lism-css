`lism.base`レイヤーの後半で定義される、トークンを再定義したり、変数セットなどを行うクラス群を `set-{className}` の形式で定義しています。

Lism では、これらを **Setter Class** と呼びます。

### `set-plain`

主要スタイルをリセットするためのクラスです。

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/base/set/_plain.scss' />

<Preview>
	<PreviewTitle>`set-plain`を`button`に適用する例</PreviewTitle>
	<PreviewArea p='20' layout=''>
		<div class='l--flex -g:20 -ff:mono'>
			<button class='set-plain'>button</button>
		</div>
	</PreviewArea>
	<PreviewCode slot='tab' label='HTML'>
	```html "'set-plain'"
	<div class='l--flex -g:20 -ff:mono'>
		<button class='set-plain'>button</button>
	</div>
	```
	</PreviewCode>
</Preview>



### `set-shadow`

`.set-shadow` クラスを指定した要素では、シャドウを構成する`--bxsh--`変数が再定義されます。  

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/base/set/_shadow.scss' />

ここで`--shc`変数を上書きすることで、シャドウの色味を調整したりすることが可能です。 

<Preview>
	<PreviewTitle>シャドウの上書き例</PreviewTitle>
	<PreviewArea p='30' nogrid>
		<Stack g='10'>
			<HTML.p> Default: </HTML.p>
			<Flex className='set-shadow -g:30' px-s='15'>
				<Box bxsh='20' p='30' bdrs='20'>Box</Box>
				<Box bxsh='40' p='30' bdrs='20'>Box</Box>
			</Flex>
			<Spacer h='40' />
			<HTML.p> set-shadow で 変数を上書きした例: </HTML.p>
			<Flex className='set-shadow -g:30' px-s='15' style={{'--shc': 'hsl(200 50% 50% / 20%)'}}>
				<Box bxsh='20' p='30' bdrs='20'>Box</Box>
				<Box bxsh='40' p='30' bdrs='20'>Box</Box>
			</Flex>
		</Stack>
		<div class='a--spacer' />
	</PreviewArea>
		<PreviewCode slot='tab' label='HTML'>
		```html "set-shadow"
		<div class="set-shadow l--flex -g:30" style="--shc:hsl(200 50% 50% / 20%)">
			<div class="l--box -bxsh:20 -p:30 -bdrs:20">Box</div>
			<div class="l--box -bxsh:40 -p:30 -bdrs:20">Box</div>
		</div>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "set-shadow"
		<Flex className='set-shadow' g='30' style={{'--shc': 'hsl(200 50% 50% / 20%)'}}>
			<Box bxsh='20' p='30' bdrs='20'>Box</Box>
			<Box bxsh='40' p='30' bdrs='20'>Box</Box>
		</Flex>
		```
	</PreviewCode>
</Preview>


### `set-bp`

特定のブレイクポイント以上でのみスタイルを適用させるためのユーティリティクラスです。
[ブレイクポイント対応](/css/responsive/)していないプロパティを変化させたい時のハック的手段として活用できます。

<EmbedCode dirPath='packages/lism-css' srcPath='src/scss/base/set/_bp.scss' />

<Preview>
	<PreviewTitle>smサイズ以上でテキストカラーを赤に変える例</PreviewTitle>
	<PreviewArea resize>
		<Box className='set-bp' p='20' style={{'color': 'var(--_is_sm) red;'}}>
			<p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。</p>
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='HTML'>
	```html "set-bp" "var(--_is_sm) red"
	<div class="set-bp l--box -p:20" style="color:var(--_is_sm) red;">
		<p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。</p>
	</div>
	```
	</PreviewCode>
	<PreviewCode slot='tab' label='JSX'>
		```jsx "set-bp" "var(--_is_sm) red"
		<Box className='set-bp' p='20' style={{'color': 'var(--_is_sm) red;'}}>
			<p>ロレム・イプサムの座り雨。目まぐるしい文章の流れの中で、それは静かに歩く仮の言葉です。</p>
		</Box>
		```
	</PreviewCode>
</Preview>

### `set-innerRs`

親要素の角丸とPaddingの値から、内側の要素の角丸(`--bdrs--inner`)を計算するユーティリティクラスです。 
親要素(`set--innerRs`)の p(`--p`), bdrs(`--bdrs`) を元に計算します。（これらの指定が必須となります。）

計算された`--bdrs--inner`をセットするには、子要素で`-bdrs:inner`を指定します。

<Preview>
	<PreviewTitle>使用例</PreviewTitle>
	<PreviewArea p='30'>
	<Lism className='set-innerRs' p='15' bd bxsh='20' bdrs='50'>
		<Frame bdrs='inner' ar='og'>
			<img src='https://cdn.lism-css.com/img/u-5.jpg' width='1920' height='1280' alt='' />
		</Frame>
	</Lism>
	</PreviewArea>
	<PreviewCode label="JSX" slot="tab">
```jsx
<Lism className='set-innerRs' p='15' bd bxsh='20' bdrs='50'>
	<Frame bdrs='inner' ar='og'>
		<img src='https://cdn.lism-css.com/img/u-5.jpg' width='1920' height='1280' alt='' />
	</Frame>
</Lism>
```
	</PreviewCode>
	<PreviewCode label="HTML" slot="tab">
```html
<div class="set-innerRs -p:15 -bd -bxsh:20 -bdrs:50">
	<div class="l--frame -bdrs:inner -ar:og">
		<img src="https://cdn.lism-css.com/img/u-5.jpg" width="1920" height="1280" alt="" />
	</div>
</div>
```
	</PreviewCode>
</Preview>
