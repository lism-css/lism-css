### `set-shadow`

`.set-shadow` クラスを指定した要素では、シャドウを構成する`--bxsh--`変数が再定義されます。  

```css
:root, .set-shadow {
	--shc: hsl(220 4% 8% / 5%);

	--bxsh--10: var(--shsz--5) var(--shc), var(--shsz--10) var(--shc);
	--bxsh--20: var(--shsz--10) var(--shc), var(--shsz--20) var(--shc);
	--bxsh--30: var(--shsz--20) var(--shc), var(--shsz--30) var(--shc);
	--bxsh--40: var(--shsz--30) var(--shc), var(--shsz--40) var(--shc);
	--bxsh--50: var(--shsz--40) var(--shc), var(--shsz--50) var(--shc);
}
```

ここで変数を上書きすることで、シャドウの色味などを調整することが可能です。

### `set-hov`

ホバースタイル管理用のCSS変数をセットします。主に子要素のホバースタイルを制御するために使用します。

```scss
@media (any-hover: hover) {
	.set-hov:hover {
		--notHov_: ;
	}
	.set-hov:not(:hover) {
		--isHov_: ;
	}
}
@media (any-hover: none) {
	.set-hov {
		--isHov_: ;
	}
}
```

<Reference>
活用例は [`hover`の解説ページ](/css/hov#set-hov) を参照してください。
</Reference>

### `set-innerRs`

親要素の角丸とPaddingの値から、内側の要素の角丸(`--bdrs--inner`)を計算するユーティリティクラスです。 
親要素(`set--innerRs`)の p(`--p`), bdrs(`--bdrs`) を元に計算します。（これらの指定が必須となります。）

計算された`--bdrs--inner`をセットするには、子要素で`-bdrs:inner`を指定します。

<Preview>
	<PreviewTitle>使用例</PreviewTitle>

		<Frame bdrs='inner' ar='og'>
			<img src='https://cdn.lism-css.com/img/u-5.jpg' width='1920' height='1280' alt='' />
		</Frame>
	</Lism>
	</PreviewArea>
	<PreviewCode label="JSX" slot="tab">
```jsx
<Lism className='set-innerRs' p='15' bd bxsh='20' bdrs='50'>
```
