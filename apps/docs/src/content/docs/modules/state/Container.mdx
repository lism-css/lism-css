[`is--wrapper`](/css/module-class/#is--wrapper) クラスをコンポーネントで利用する例を紹介していきます。

`is--wrapper` は基本的に `isWrapper`プロパティで**どのコンポーネントからでも呼び出せるようになっています。**

| プロパティ | 出力 |
|---|---|
| `isWrapper`| `.is--wrapper` |
| `isWrapper='s'` | `.is--wrapper`, `.-content:s` |
| `isWrapper='l'` | `.is--wrapper`, `.-content:l` |
| `isWrapper={value}` | `.is--wrapper`, `--contentSize:{value}` |


また、`isContainer` (`is--container`)クラスと併用することで、`container-type: inline-size` も適用できます。


## 専用コンポーネント

`<Lism isWrapper>` のエイリアスとして、`<Wrapper>` も用意しています。


### Props
| プロパティ | 説明 |
|---|---|
| `contentSize` | コンテンツサイズを指定します。|
| `layout` | 展開したい Layoutモジュールを指定できます。|
| `isContainer` | コンテナタイプをセットでいます。|

<PreviewTitle>下記の ①, ② はそれぞれ同じ結果になります。</PreviewTitle>

```jsx
// ①
<Flow isWrapper='s' isContainer>...</Lism>

// ②
<Wrapper contentSize='s' layout='flow' isContainer>...</Wrapper>
```

### 任意のコンテナサイズをセットする

s, l 以外の値をセットしたい場合は、isWrapper に任意の値をセットすると`--contentSize`変数としてしゅつ力されます。

<Preview>
	<PreviewTitle>is--wrapper に任意の値をセットする</PreviewTitle>
	<PreviewArea resize p='15'>
		<Lism isWrapper='20rem' p='20' className='-bd:guide -hov:itemGuide'>
			<div>Contents...</div>
			<div>Contents...</div>
		</Lism>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
	```jsx
	<Lism isWrapper='20rem' p='20'>
		<div>Contents...</div>
		<div>Contents...</div>
	</Lism>
	```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
	```html
	<div class="is--wrapper -p:20" style="--contentSize:20rem">
		<div>Contents...</div>
		<div>Contents...</div>
	</div>
	```
	</PreviewCode>
</Preview>
