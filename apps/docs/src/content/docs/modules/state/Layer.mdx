{/* [`is--container`](/css/module-class/#is--container) */}
[`.is--layer`](/css/module-class/#is--layer)をデフォルトで出力するコンポーネント(`<Lism isLayer>`のエイリアス）です。



{/* <Reference>
要素を単純に`position:absolute`で配置したいだけの場合は、`-pos:abs`クラスなどを使用することもできます。
参考：[absoluteで配置する方法](/components/props/others/absolute)
</Reference> */}


## Props

| プロパティ | 説明 |
|---|---|
| `blur`,`contrast`,`brightness`,`grayscale`,`hueRotate`,`invert`,`saturate`,`sepia` | これらをまとめて連結し、 `style.backdropFilter` に受け渡します。|

<ImportPackage component='Layer' />

### backdrop-filterの活用

`<Layer>`コンポーネントでは、`backdrop-filter`で使用できる`blur`などの主要成分を個別に指定できるように追加調整されています。

<Preview>
	<PreviewTitle>背景にブラーをかける</PreviewTitle>
	<PreviewArea p='20'>
		<Box pos='rel' p='40'>
			<HTML.p fz='2xl' fw='bold' ta='center'>BACKGROUND</HTML.p>
			<Layer blur='3px' />
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx {3}
		<Box pos='rel' p='40'>
			<HTML.p fz='2xl' fw='bold' ta='center'>BACKGROUND</HTML.p>
			<Layer blur='3px' />
		</Box>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html {3}
		<div class="l--box -pos:rel -p:40">
			<p class="-fz:2xl -fw:bold -ta:center">BACKGROUND</p>
			<div class="is--layer" style="backdrop-filter:blur(3px)"></div>
		</div>
		```
	</PreviewCode>
</Preview>

### メディアレイヤー

`.is--layer`を使ってコンテンツの背景に画像を配置するようなレイアウトを組んでみます。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea p='20'>
		<Box hasGutter pos='rel' py='50'>
			<Frame isLayer>
				<Media src="/img/img-3.jpg" alt="" width='960' height='640' loading='lazy' decoding='async' />
			</Frame>
			<Layer bgc='black:50%' />
			<Stack pos='rel' g='40' c='white'>
				<Dummy/>
				<Dummy length='l' />
			</Stack>
		</Box>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx {2-4} "\<Frame isLayer"
		<Box hasGutter pos='rel' py='50'>
			<Frame isLayer>
				<Media src="/img/img-3.jpg" alt="" width='960' height='640' loading='lazy' decoding='async' />
			</Frame>
			<Layer bgc='black:50%' />
			<Stack pos='rel' g='40' c='white'>
				<Dummy/>
				<Dummy length='l' />
			</Stack>
		</Box>
		```
	</PreviewCode>
	<PreviewCode slot='tab' label='HTML'>
		```html {2-4}
		<div class="l--box has--gutter -py:50">
			<div class="l--frame is--layer">
				<img class="a--media" src="/img/img-3.jpg" width="960" height="640" loading="lazy" decoding="async">
			</div>
			<div class="is--layer" style="--bgc:color-mix(in srgb, var(--black) 50%, transparent)"></div>
			<div class="l--stack -pos:rel -c -g:40" style="--c:var(--white)">
				<p>Lorem ipsum dolor sit amet...</p>
				<p>Lorem ipsum dolor sit, amet...</p>
			</div>
		 </div>
		```
	</PreviewCode>
</Preview>
