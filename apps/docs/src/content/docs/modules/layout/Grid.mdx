import { Grid, Lism, Box, Center, Decorator, Frame, Media, Dummy } from 'lism-css/astro';
import { Preview, PreviewArea, PreviewTitle, PreviewCode, ImportPackage, PropBadge, Callout } from "~/components";


コンテンツを**Gridレイアウトで配置する**ためのコンポーネントです。


## Import

<ImportPackage component='Grid' />


## Props

|Prop|CSS|
|---|---|
|`gd`|`grid`|
|`gt`|`grid-template`|
|`gta`|`grid-template-areas`|
|`gtc`|`grid-template-columns`|
|`gtr`|`grid-template-rows`|
|`gaf`|`grid-auto-flow`|
|`gar`|`grid-auto-rows`|
|`gac`|`grid-auto-columns`|

### `-gtc:liquid`

`l--grid` との併用が前提の、`gtc`ユーティリティクラスです。

列数を指定せず、**子要素が指定した幅より小さくならないように自動で折り返しされる**カラムレイアウトを作成します。

#### ペアプロパティ

`cols`で子要素が維持する最小幅を指定できます（デフォルトは`--sz--min`です）。

<Preview>
	<PreviewTitle>デフォルト幅</PreviewTitle>

			<Box bxsh='20' p='20'><Dummy/></Box>
			<Box bxsh='20' p='20'><Dummy offset='1' /></Box>
			<Box bxsh='20' p='20'><Dummy offset='2' /></Box>
			<Box bxsh='20' p='20'><Dummy length='s' /></Box>
			<Box bxsh='20' p='20'><Dummy /></Box>
		</Grid>
	</PreviewArea>
	<PreviewCode>
		```jsx
		<Grid gtc='liquid' g='20'>
			<Box bxsh='20' p='20'><Dummy/></Box>
			<Box bxsh='20' p='20'><Dummy/></Box>
			<Box bxsh='20' p='20'><Dummy offset='1' /></Box>
		```

### `-gt:repeat`

`cols`で列数、`rows`で行数を指定できる`gt`ユーティリティクラスです。

それぞれ、ブレイクポイント指定も可能で初期値は `1` です。


<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea resize p='20'>
		<Grid gt='repeat' cols='3' rows='3' g='5' min-h='50svh'>
			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
			<Box p='20' bgc='base-2' gc='span 2'>item5</Box>
			<Center gc='3' gr='1 / -1' p='20' bgc='blue'>item6</Center>
		</Grid>
	</PreviewArea>
	<PreviewCode slot='tab' label='JSX'>
		```jsx " gc=" " gr="
		<Grid gt='repeat' cols='3' rows='3' g='5' min-h='50svh'>
			<Box p='20' bgc='base-2'>item1</Box>
			<Box p='20' bgc='base-2'>item2</Box>
			<Box p='20' bgc='base-2'>item3</Box>
			<Box p='20' bgc='base-2'>item4</Box>
			<Box p='20' bgc='base-2' gc='span 2'>item5</Box>
			<Center gc='3' gr='1 / -1' p='20' bgc='blue'>item6</Center>
		</Grid>
		```
	</PreviewCode>
</Preview>


cols,rows はそれぞれブレイクポイント指定も可能です。
