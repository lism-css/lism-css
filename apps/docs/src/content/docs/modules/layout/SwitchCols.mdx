## Props

| プロパティ | 説明 |
|---|---|
| `breakSize` <PropBadge type="cssvar">`--breakSize`</PropBadge> | 横並びを維持するのに必要な幅（親のサイズ）を指定できます。[szトークン](/css/tokens/#size)の指定ができます。|
