|---|---|

### `Grid`でアイコンを端に寄せる

### `hov="reverse"`

ホバー時に、fill,outlineが入れ替わるような [`-hov`クラス](/css/hov)を追加する例を紹介します。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea p='20' bg=':stripe' layout=''>
		<Flex g='15'>
			<Button href='###' hov='reverse' variant='fill'>Button</Button>
			<Button href='###' hov='reverse' variant='outline'>Button</Button>
		</Flex>
	</PreviewArea>

	<PreviewCode slot='tab' label='追加CSS'>
	```scss
	.-hov\:reverse {
		--hov-c: var(--bgc);
		--hov-bgc: var(--c);
		&:where(.c--button--fill) {
			--hov-bgc: transparent;
		}
		&:where(.c--button--outline) {
			--hov-c: var(--base);
		}
		@media (any-hover: hover) {
			&:hover {
				background-color: var(--hov-bgc) !important;
				color: var(--hov-c) !important;
			}
		}
	}
	```
	</PreviewCode>
