当サイトは、CSSに関する個人的な知見をアウトプットするためのブログです。

ドキュメントサイト風のレイアウトを採用しており、CSSに関して体系的に学べるサイトへと発展させていきたいと考えています。


- 自分のプロジェクトを作る上で培った知見
- 過去に自分で書いていたブログ記事をアップデートさせたもの
- その他ネットで集めた知見をもとに自分なりに整理した情報

これらに基づいて、当サイトを運営していきます。

なかなかブログ記事として情報を発信するのは大変なので腰が重くて長いことできていませんでしたが、AIが人様に向けて発信しても問題ない文体へサッと整えてくれるようになったので、それを活用していきたいと思います。


## 管理人

WordPressテーマを開発・販売したりしているフロントエンドエンジニアです。

以前は他のフロントエンド系テックブログを運営していましたが、内容がほとんど古くなってきたため閉鎖し、新しく当サイトを立ち上げました。


## Lism CSS

このサイトで紹介しているような知見も含めて、個人的にCSS設計について体系的に整理したものを [Lism CSS](https://lism-css.com) としても公開しています。

https://lism-css.com

まだβ版ですので仕様変化が激しいですが、ぜひご覧いただければと思います。



## AIによる執筆について

基本的には自分で書いており、コードの内容は、すべて自分で検証した上でアウトプットしたものです。

ただし、文章の肉付けや体裁を整える編集作業に関しては一部AIにも手伝ってもらうこともあります。  
（AIで生成・編集した部分に関しては、その結果を再度自分で確認・修正した上で公開していきます。）
