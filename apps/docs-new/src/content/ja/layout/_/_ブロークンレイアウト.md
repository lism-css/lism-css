# ブロークンレイアウト - レイアウトをあえて崩して視線を惹きつける

ブロークンレイアウトは、整ったグリッドをあえて崩して視線を引く手法です。ただ崩しすぎると読みにくさや破綻を招きます。本セクションでは、Gridを土台に「どこを崩し、どこを揃えるか」を押さえた安全な作り方を、実例コードとともに解説します。


## ブロークンレイアウトの全体像を把握する

ブロークンレイアウトは、画像とテキストがきれいな四角形のグリッドや左右均等レイアウトから意図的に外れて配置されている点にあります。ここでは、あえて「整列しない位置」に要素を置くことで視線の流れを崩し、デザインに動きをつくるのが特徴です。

まず、サンプル全体のコードを掲載します。この後で、どの指定が"ブロークン"を生み出しているのかをステップごとに見ていきましょう。

```css
.section {
  max-inline-size: var(--content-size, 1200px);
  margin-inline: auto;
  padding: var(--space--6);
}

.img-frame > img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

/* ブロークンレイアウト-01 */
.broken-01 {
  display: flex;
  flex-direction: column;
  gap: var(--space--8);
}

/* コンテンツ */
.broken-01__content {
  /*  親に対する自身の配置  */
  align-self: center;

  /*  コンテンツレイアウト  */
  display: flex;
  flex-direction: column;
  gap: var(--space--4);
}
.broken-01__content > h2 {
  font-size: var(--font-size--2xl);
  font-weight: 200;
  letter-spacing: 0.025em;
}
.broken-01__content > p {
  font-size: var(--font-size--sm);
}

/* 画像をグリッドレイアウトで配置 */
.broken-01__images {
  display: grid;
  grid-template-columns: repeat(4, 1fr);

  flex-shrink: 0;

  grid-column: span 2 / -1;
  grid-row: 1 / -2;
}

  grid-column: 1 / span 2;
  grid-row: 2 / -1;
}

/* 画面が広い時は .broken-01__content と .broken-01__images が横並び */
@media (min-width: 800px) {
  .broken-01 {
    flex-direction: row;
  }
  .broken-01__images {
    width: 40vw;
  }
}
```

### レイアウト全体のキャンバスをつくる（.section）

`.section`で最大幅（`max-inline-size`）と左右中央寄せ（`margin-inline: auto`）を指定し、1セクション分の「キャンバス」を用意しています。ここにテキストブロックと画像ブロックが乗るイメージです。

```css
.section {
  max-inline-size: var(--content-size, 1200px);
  margin-inline: auto;
  padding: var(--space--6);
}
```


### テキストと画像を縦積み→横並びに切り替える（.broken-01）

`.broken-01`は`display: flex; flex-direction: column;`で、「テキスト→画像」の縦並びレイアウトになります。画面が広くなったら @media内で`flex-direction: row;`に切り替え、テキストと画像が左右に並ぶ構成に変わります。

```css
.broken-01 {
  display: flex;
  flex-direction: column;
  gap: var(--space--8);
}
```


### テキストブロックを中央寄せして"崩した位置"に置く（.broken-01__content）

`.broken-01__content`では`align-self: center;`により、親flexの中でテキストブロック自体をやや中央寄せに配置しています。これにより、テキストがコンテナの左端に「きれいに」揃わず、画像との位置関係にわずかなズレが生まれます。

```css
.broken-01__content {
  align-self: center;
  display: flex;
  flex-direction: column;
  gap: var(--space--4);
}
```


### 画像側に大きなグリッドキャンバスを用意する（.broken-01__images）

`.broken-01__images`は`display: grid;`と`grid-template-columns: repeat(4, 1fr);`・`grid-template-rows: repeat(6, 1fr);`により、4×6のマス目を持った「大きな画像キャンバス」を作っています。

`width: 80vw; min-height: max(50vw, 40vh); margin-left: auto;`によって、画面幅に応じて大きく広がりつつ、全体の右側に寄せられます。ここが「大胆に画像を広げる」ための土台部分です。

```css
.broken-01__images {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(6, 1fr);
  gap: var(--space--2);
  flex-shrink: 0;
  width: 80vw;
  min-height: max(50vw, 40vh);
  margin-left: auto;
}
```


### 2枚の画像を"きれいに揃えず"ずらして重ねる（.broken-01__img-1, .broken-01__img-2）

2枚の画像をずらして重ねます。ここが最大のポイントとなります。1枚目は右上〜右下に大きく配置し、2枚目は左下〜中央に重なり気味に配置します。

この「対角線上のズレ／重なり」が「四角い整列レイアウトではありえない視線の動き」を生み出し、ブロークンレイアウトらしいダイナミックな印象が生まれます。

```css
.broken-01__img-1 {
  grid-column: span 2 / -1;
  grid-row: 1 / -2;
}
```

```css
.broken-01__img-2 {
  grid-column: 1 / span 2;
  grid-row: 2 / -1;
}
```


### 画面が広い時に"テキスト＋ブロークン画像"の対比を強調する（@media部分）

`@media (min-width: 800px)` では`.broken-01`を横並び（row）にし、`.broken-01__images`の幅を40vwに抑えています。これにより、左側にテキスト、右側に「ずらして重ねた画像の塊」という構図がはっきり現れ、ヘッダー下のセクションなどで印象的なヒーローブロックとして使いやすいレイアウトになります。

```css
@media (min-width: 800px) {
  .broken-01__images {
    width: 40vw;
  }
}
```


## 画像を対角線にずらしたブロークンレイアウト

ブロークンレイアウトの作例として画像を対角線にずらしたレイアウトを紹介します。主なポイントは以下のとおりです。

- 小さい画面ではFlex＋マイナスmarginで画像を重ねてズラす → 四角いカード型の流れから外れ、構図にアクセントが生まれる
- 大きい画面では24×24のGridに切り替え、画像・タイトル・テキストを斜め方向に配置 → "整って見えるのに整列していない"絶妙な違和感を演出
- 画像1・画像2・タイトル・本文がそれぞれ異なる位置に散らばるようデザイン → 一見ランダムでも、Grid上ではきれいに管理されている

以下の作例は、画像とテキストを左右・上下でそろえずに"対角線方向へズラす"タイプのブロークンレイアウトです。きれいな2カラムや整列したカードではなく、「あえて重なり・ズレ」を作ることで動きと奥行きを出すのがポイントです。

```css
.section {
  max-inline-size: var(--content-size, 1200px);
  margin-inline: auto;
  padding: var(--space--6);
}
.img-frame {
  overflow: hidden;
}
.img-frame > img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

/* ブロークンレイアウト-02 */
.broken-02 {
  display: flex;
  flex-direction: column;
}

.broken-02__title {
  font-size: var(--font-size--2xl);
  font-weight: 200;
  line-height: 1.2;
  letter-spacing: 0.025em;
}
.broken-02__subtext {
  font-size: var(--font-size--sm);
}
.broken-02__img-1 {
  box-shadow: var(--shadow--md);

  /*  小さい画面サイズでの配置  */
  aspect-ratio: 21 / 9;
  width: 45%;
  margin-top: 1.5em;
  margin-left: auto;
}
.broken-02__img-2 {
  box-shadow: var(--shadow--lg);

  /*  小さい画面サイズでの配置  */
  aspect-ratio: 3 / 2;
  width: 70%;
  margin-top: -4vw;
  margin-bottom: 2em;
}

/* 画面が広い時 */
@media (min-width: 800px) {
  .broken-02 {
    display: grid;
    grid-template-columns: repeat(24, 1fr);
    grid-template-rows: repeat(24, 1fr);
    height: max(20em, 80vh);
  }
  .broken-02 > .img-frame {
    width: auto;
    margin: 0;
    aspect-ratio: unset;
  }
  .broken-02__img-1 {
    grid-column: span 12 / -1;
    grid-row: 1 / span 8;
  }
  .broken-02__img-2 {
    grid-column: 1 / span 15;
    grid-row: 8 / -1;
  }
  .broken-02__title {
    grid-column: 2 / span 10;
    grid-row: 2 / span 5;
    align-self: center;
  }
  .broken-02__subtext {
    grid-column: span 8 / -1;
    grid-row: span 14 / -1;
    place-self: end;
  }
}
```

## 縦帯の写真と巨大タイポで見せるヒーローブロークン

続いて画面全体を使ったヒーローセクションを「細い写真の帯＋巨大なタイトルタイポ」で崩したブロークンレイアウトを紹介します。

「左に細い写真・中央に巨大な文字・右下に短いテキスト」という、きれいな2カラムや中央揃えヒーローから意図的に外したレイアウトになっています。

主なポイントは以下のとおりです。

- `.broken-03`を24×24のgridにし、ヒーローエリア全体を細かいマス目として扱う
- `.broken-03__img` を左端 6カラム分だけに縦長に配置し、`::after`の`backdrop-filter`でブラーをかけて"背景のノイズ帯"のように見せる
- `.broken-03__title`を`grid-column: 3 / -3; grid-row: 4 / -4;`で中央寄りに大きくまたいで配置し、通常の「画像の下に見出し」構造から大きくはみ出させる
- `.broken-03__subtext`は右下寄せで`place-self: end;`とし、タイトルとは別のラインにずらして置いて、視線の動きをジグザグにしている

```css
.img-frame > img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

/* ブロークンレイアウト-03 */
.broken-03 {
  display: grid;
  grid-template-columns: repeat(24, 1fr);
  grid-template-rows: repeat(24, 1fr);
  height: 100svh;
  padding: 0.75em;
}
.broken-03__img {
  grid-column: 1 / span 6;
  grid-row: 1 / -1;
  z-index: 0;
  position: relative;
}
.broken-03__img::after {
  content: "";
  position: absolute;
  inset: 0;
  backdrop-filter: blur(8px) grayscale(0.4);
}
.broken-03__title {
  grid-column: 3 / -3;
  grid-row: 4 / -4;
  align-self: center;
  z-index: 1;
  line-height: 1;
  font-weight: 800;
  letter-spacing: 0.05em;
  font-size: min(18vw, 16.6vh); /* 大きい画面サイズでは 24マス分の4つ分の高さ */
  text-shadow: -0.5em 0 4em var(--color--bg);
}
.broken-03__title > h2 {
  line-height: 1;
  font-weight: 800;
  letter-spacing: 0.05em;
  font-size: min(16vw, 16.6vh);
}
.broken-03__subtext {
  grid-column: 10 / -2;
  grid-row: span 3 / -2;
  place-self: end;
  max-width: 20em;
  font-size: 1em;
  opacity: 0.8;
  z-index: 1;
}

/* 画面が小さい時 */
@media (max-width: 500px) {
  .broken-03__title {
    grid-column: 2 / -2;
    grid-row: 2 / -6;
  }
}
```

```html
<div class="hero-container">
  <div class="broken-03">
    <div class="img-frame broken-03__img">
      <img src="https://book-cdn.pages.dev/img/img-1.jpg" alt="" width="1200" height="800">
    </div>
    <h2 class="broken-03__title">Section<br />Title</h2>
    <p class="broken-03__subtext">ロレム・イプサムの座り雨、トマ好き学習だったエリット、しかし時と活力はそのような木々と楽しみ。</p>
  </div>
</div>
```

詳しく分解すると説明が長くなるので、細かなGridの指定はここでは割愛しますが、これまで紹介してきた「行・列ラインの指定・spanの考え方・重ね置きのテクニック」をそのまま応用した例になっています。コードを眺めながら、「どの要素をどのマス目に置いているのか？」をトレースしてみると、ブロークンでも"中身はきれいに管理されたGridであることが実感できるはずです。

## 画像回り込みレイアウト

グリッドレイアウトではありませんが、ブロークンレイアウトのもう一つの手法として、`shape-outside`を使ったテキストの回り込みについて紹介します。

画像を`clip-path`で切り抜き、かつ`float`でテキストを回り込み表示させている時に`shape-outside`に対して`clip-path`と同じパスを指定しています。これによって切り抜いた画像の形にそってテキストを回り込ませることで、紙面デザインのような動きのある配置が実現できます。

コード自体はやや複雑になるため詳細な解説は割愛しますが、これまでの「整列から外す」「形をずらす」という考え方と同じ発想で作られています。興味があれば、ぜひサンプルコードを眺めながら試してみてください。

なお、ここで出てくる`clip-path`や`mask`などについては、Chapter3でもう少し詳しく紹介します。

```css
.img-frame > img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

.shape-outside-content {
  display: flow-root;
}
.shape-outside-content > .img-frame {
  /* 変形回り込み */
  shape-outside: var(--clip-path, unset);
  clip-path: var(--clip-path, unset);
}
.img-frame--circle {
  float: left;
  width: min(30vw, 300px);
  aspect-ratio: 1 / 1;
  margin-right: 1.5em;
  margin-bottom: 1.5em;

  /* 円形に回り込み */
  --clip-path: circle(50%);
}
.img-frame--polygon {
  float: right;
  width: min(30vw, 300px);
  aspect-ratio: 1 / 1;
  margin-left: 1.5em;
  margin-bottom: 1.5em;

  /* 多角形に回り込み */
  --clip-path: polygon(0 0, 100% 0, 100% 100%, 25% 100%);
}
```

## まとめ

- **あえて崩して視線を誘導するブロークンレイアウト**: 四角い整列から外し、ズレや重なりで視線の流れを演出する。
- **Gridの土台で安全に崩すブロークンレイアウト**: Gridを土台にし、行列のマスで管理しつつ大胆な崩し表現を行える。
- **整列と余白で読みやすさを保つブロークンレイアウト**: 整列させる要素と崩す要素を分け、余白設計で読みやすさとデザイン性を両立できる。
